# 基于柔性触觉感知的术中肺结节定位探索：从结节探测到大小反演与粗深度辨别

## 摘要
**目的**：微创胸腔镜和机器人辅助手术削弱了术者在肺结节切除中的直接触觉反馈，导致小结节、深部结节和隐匿性结节的术中定位更加困难。本文关注的不是一个预设好的三层反演任务，而是一个更自然的探索过程：柔性触觉时空信号能否首先支持可靠结节探测，并在此基础上进一步提供结节大小和粗深度信息。

**方法**：本文基于 7 个结节大小、6 个埋藏深度和 3 次重复构建离体猪肺触觉实验矩阵。研究首先结合相关文献、触觉机制分析和结构化物理特征提取，使用 XGBoost 逐层验证三个问题：是否存在结节、结节大小、以及粗深度是否可辨识。随后，进一步设计仅输入原始时空张量的层级神经网络，分别完成结节检测、大小反演和大小条件下的粗深度分类。最后，通过 SHAP、latent probe、Integrated Gradients、hard-pair analysis 与 phase occlusion 等解释性分析，检验模型是否利用了与前述物理分析一致的关键结构。

**结果**：XGBoost 基线首先证明，柔性触觉数据中存在稳定的检测信息、较强的大小信息以及可学习但更弱的粗深度信息。进一步地，raw-input 检测模型在独立测试集上取得 `AUC = 0.8383`，高于 XGBoost 的 `0.8199`；raw-input size-only 模型取得 `Top-1 = 0.7177`、`Top-2 = 0.8195` 和 `MAE = 0.1242 cm`，全面优于 XGBoost 的 `0.6701`、`0.8023` 和 `0.1472 cm`。在深度任务上，普通共享深度头会退化至多数类附近，而在 `size-aware + route-aware` 组织下，纯 raw-input 路线的 predicted-route 平衡准确率达到 `0.5240`，超过 XGBoost 的 `0.5138`。解释性分析进一步显示，当前 route-aware 深度模型的 latent probe 平均测试 `R^2 = 0.2712`，高于 size-only probe 的 `0.2356`；Integrated Gradients 与 hard-pair 分析表明模型确实利用了部分与深度相关的空间结构，但 phase occlusion 也表明其时间策略仍偏向 `peak neighborhood`。

**结论**：柔性触觉时空数据首先能够支持可靠的术中结节探测，并在此基础上支持大小反演和粗深度辨别。深度并非完全不可学，但其建模必须显式尊重 `size-depth` 耦合，而不能简单视为与大小平级的共享输出。

**意义**：本文的价值不在于宣称已完成定型化临床系统，而在于建立了一条更符合 biomedical engineering 研究逻辑的证据链：从问题发现、机制分析和结构化基线出发，逐步过渡到 raw-input 深度学习与解释性验证，并进一步形成可部署的术中系统原型。

**关键词**：肺结节探测；柔性触觉阵列；术中导航；大小反演；粗深度分类；可解释人工智能

## 1 引言
随着低剂量 CT 筛查的普及，临床上对微小肺结节和磨玻璃结节的检出率持续上升。对于这类疑似早期病灶，胸腔镜手术和机器人辅助手术因创伤小、恢复快而被广泛采用。然而，微创获益伴随一个长期存在但尚未被充分解决的问题：术者在经由细长器械操作时，无法像开放手术那样直接触摸肺组织，从而失去对结节位置、大小和边界的重要触觉判断。对于深部、小尺寸或表面不明显的结节，这种触觉缺失会显著增加术中定位难度。

目前的常用辅助方案包括术前 CT 引导钩线或染料定位、术中超声以及有限度的数字触诊等 [R1-R3]。这些方法各自具有明确价值，但也存在侵入性风险、空气干扰、操作依赖强或主观性高等不足。对术中定位而言，真正缺失的并不是“又一种视觉信息”，而是一种能够在微创条件下恢复机械触感、并将其转化为客观定量判断的技术路径。

在微创触觉恢复方向，已有机器人辅助触觉感知、力反馈融合、声反射式触觉传感和相关综述工作表明，深部包块的存在能够通过表面机械响应差异被感知 [R4-R8]。这些研究为本文提供了重要出发点。但回到肺结节这一特定场景，我们认为一个更关键的问题是：如果用柔性阵列连续记录肺表面在动态按压中的二维应力分布，那么这些时空信号究竟能支持什么程度的反演？

本文的工作并不是一开始就把“是否有结节、结节大小、结节深度”三项内容并列为同等地位的既定任务。相反，问题是随着实验设计和数据观察逐步展开的。实验矩阵天然包含两个真实物理变量：结节大小和结节深度。首先，我们需要回答的是最直接的临床问题，即**当前位置下方是否存在结节**。只有当这一点成立之后，才值得继续问：这些信号是否还稳定携带了与**结节大小**相关的信息。再进一步，考虑到深度效应通常弱于大小效应，且与大小、形态、扩散和按压阶段等多种因素耦合，我们才谨慎地把**粗深度辨别**作为后续的探索性问题。

这一点决定了本文的研究路径。我们采用的不是“先设定三层输出、再寻找一个统一黑箱模型”的写法，而是一条更渐进的路线：
1. 从文献、机制分析和结构化特征出发，先判断数据里是否存在与结节存在、大小和深度相关的可学习线索；
2. 使用 XGBoost 建立结构化基线，逐层验证 detection、size 和 coarse depth 是否可学；
3. 在此基础上，再构建仅输入原始时空张量的层级神经网络，检验原始输入本身能否自动学习这些逐步发现的结构；
4. 最后通过解释性分析与部署验证，确认网络到底学到了什么，以及这些结论在系统层面是否具有工程意义。

在这个框架下，XGBoost 并不是一个等待被替换掉的“旧方法”，而是本文证据链中的第一个关键环节。它的作用至少有三点。第一，它是结构化机制基线，用于回答“这些信息客观上是否存在且可学”；第二，它是解释锚点，用于识别哪些特征家族对结节存在、大小和深度最重要；第三，它为后续 raw-input 神经网络提供了一个清晰参照，使我们能够区分“神经网络是否真正从原始张量学到了东西”与“它是否只是借助某种捷径重现了部分表面相关性”。

基于上述认识，本文希望系统回答以下四个问题：
1. 柔性触觉时空数据能否首先支持稳定、可靠的术中结节探测？
2. 在探测可行之后，信号中是否还存在可反演的大小信息？
3. 对于更弱且更耦合的深度变量，是否能够在更保守的 coarse depth 层面获得可辨识性？
4. 如果 raw-input 神经网络在这些任务上达到甚至超过结构化基线，那么它内部是否确实编码了与前述物理特征一致的关键结构？

围绕这些问题，本文的主要贡献可以概括为：
1. 构建了一个适合渐进式问题展开的触觉实验体系，在 7 个大小、6 个深度和 3 次重复条件下，形成了面向 detection、size 和 coarse depth 的统一窗口级数据协议。
2. 基于文献启发、触觉机制分析与结构化物理特征，建立了 XGBoost 基线，逐层验证了结节探测、大小反演和粗深度辨别的可行性，并给出对应特征家族的解释。
3. 提出了以 `Detection -> Size -> Depth` 为核心组织方式的 raw-input 层级神经网络，并证明原始时空触觉张量本身足以在关键任务上达到甚至超过 XGBoost。
4. 通过 SHAP、latent probes、Integrated Gradients、hard-pair analysis 与 phase occlusion 建立了解释性闭环，说明神经网络确实自动编码了与关键物理因素一致的部分结构。
5. 在 scientific main line 之外，进一步构建统一层级反演器与实时界面原型，验证了该路线在真实 predicted-route 与在线使用场景下的工程潜力。

## 2 材料与方法

### 2.1 问题定义与研究路径
本文关注的是一个由临床需求逐步展开的触觉反演问题，而不是一个预设好的多头分类问题。任务的出现顺序如下：
1. **结节探测**：当前按压位置下方是否存在结节；
2. **大小反演**：在确认结节存在后，触觉信号是否足以估计结节大小；
3. **粗深度辨别**：在大小和深度同时存在的实验设计下，深度是否能够被更保守地分为 `shallow / middle / deep` 三类。

其中，结节大小和结节深度是实验设置中真实存在的两个物理变量；深度问题并不是独立于大小自动产生的强信号，而是一个更弱、更耦合的条件效应。因此，本文并不将其写成连续深度反演，而是明确界定为 `coarse depth discrimination`。

### 2.2 实验平台与数据协议
本文使用柔性阵列触觉传感系统对离体猪肺模型进行动态按压实验。每帧包含 96 个传感通道，经重排后形成 `12 × 8` 的二维应力矩阵。实验矩阵由 7 个结节大小和 6 个埋藏深度构成，共 42 个物理条件；每个条件进行 3 次独立重复，共获得 126 组带完整时间演化信息的触觉记录。

模型采用长度为 10 帧、步长为 2 的滑动窗口作为基本样本单位。检测任务的标签由人工标注阳性片段决定，窗口中心帧位于阳性片段时记为正窗；大小和深度标签则直接继承自实验条件。检测任务采用 `1.CSV + 2.CSV` 开发、`3.CSV` 测试的协议；大小与深度任务采用更严格的 `1.CSV` 训练、`2.CSV` 验证、`3.CSV` 测试协议，并仅在真实阳性窗口上训练。

### 2.3 结构化物理特征与 XGBoost 基线
在深度学习之前，本文首先从原始触觉序列中提取具有物理意义的结构化特征。其来源并不是简单地“照搬文献”，而是由三部分共同决定：
1. 现有微创触觉与病灶定位文献对机械响应的启发；
2. 本实验中大小和深度变量对表面应力场影响的机制判断；
3. 原始数据中可稳定观察到的峰值、扩散、形态和阶段响应差异。

最终使用的特征大致可归纳为五类：
1. `amplitude response`：峰值、积分响应、整体能量等；
2. `spread extent`：热点面积、热点半径、二阶矩扩散等；
3. `shape contrast`：中心-边缘对比、空间熵、形态复杂度等；
4. `deformation position`：质心位置及漂移等；
5. `temporal phase`：峰值到达时间、上升/衰减趋势及阶段统计量等。

在此基础上，本文使用 XGBoost 分别完成 detection、size classification / regression 和 coarse depth classification。其角色不是最终部署，而是用来逐层回答“这个任务是否可能”。

### 2.4 基于 SHAP 的结构化解释
为解释 XGBoost 的判别依据，本文进一步使用 SHAP 分析不同任务下最重要的特征家族。检测任务主要依赖明显的强度与集中度差异；大小任务主要依赖峰值、积分响应及中心-边缘对比；而深度任务则不再主要由单一峰值幅值决定，而更多依赖扩散、形态、形变位置和时间阶段等复合结构。这个结果对后续神经网络设计具有直接指导意义。

### 2.5 纯 raw-input 层级神经网络

#### 2.5.1 阶段一：结节探测
阶段一检测器直接从原始时空触觉窗口判断结节是否存在。输入包括原始幅值窗口和相邻帧差分窗口，空间特征由轻量卷积编码，时间特征由 MS-TCN 聚合，最后输出结节存在概率。这个阶段的目标很明确：先建立稳定可靠的 detection mainline，而不是一开始就同时估计全部内部参数。

#### 2.5.2 阶段二：大小反演
在真实阳性窗口上，本文进一步训练 raw-input size-only router v2，同时输出 7 类大小分类和连续大小估计。该模型采用 `size classification + ordinal size prediction + expectation-based continuous estimation + residual correction` 的组合结构，以同时兼顾档位判断和档内连续修正。由于大小是强主效应，同时又是深度建模的路由条件，因此它被单独建模，而不是简单附属于共享多任务头。

#### 2.5.3 阶段三：大小条件下的粗深度分类
深度任务并未被建模为一个普通共享头，而是被组织为 `size-aware` 的 routed experts。其核心原因是：深度效应本身较弱，而且会被大小这一更强因素掩盖。如果仍要求网络在全局共享特征上直接分出深度类别，模型往往会优先吸收大小和强度差异，导致深度退化。为此，本文将深度建模为在大小条件下进行的 `shallow / middle / deep` 三分类，并进一步通过 `GT route / hard predicted route / soft route / top2 route` 的联合训练实现 `route-aware` 优化。

### 2.6 raw-input 神经网络的可解释性分析
为了验证 raw-input 神经网络是否真正学到了与大小和深度相关的结构，本文对当前 route-aware depth v2 模型执行四类解释性分析：
1. **latent linear probes**：在冻结模型后，用潜在表示恢复前述结构化特征，检验网络是否自动编码了这些概念；
2. **Integrated Gradients**：在正确预测测试窗口上计算输入归因，并按深度类别聚合为平均空间归因图；
3. **hard-pair analysis**：构造“大而深 / 小而浅”等容易混淆的样本对，观察模型是否仍能在减弱峰值差异后给出更合理判断；
4. **phase occlusion**：按 `loading early / loading late / peak neighborhood / release` 分阶段遮挡输入，检验模型在时间维上真正依赖什么。

### 2.7 面向部署的统一层级反演器与 GUI
在 scientific main line 之外，本文还构建了部署导向的统一层级反演器。该模型联合输入原始幅值、归一化形态、差分序列和结构化物理特征，并同时优化 size 与 routed depth，以增强真实 predicted-route 条件下的系统鲁棒性。最终，阶段一检测器与统一层级反演器被整合到实时 GUI 中，界面持续显示 detection 结果，而 size 与 depth 仅在高置信条件下输出。

## 3 结果

### 3.1 任务是逐层被发现的，而不是一开始就被并列设定的
结构化特征分析首先揭示了一个明确的信息层级：检测信号最强，大小是显著主效应，而深度是更弱且易被大小掩盖的条件效应。也就是说，实验数据并没有自然支持一个“三头并列”的对称任务定义。相反，它更支持一条由强到弱、由直接到间接的问题展开路径：先 detection，再 size，最后是更谨慎的 coarse depth。

### 3.2 XGBoost 基线先证明了“这个问题是可能的”
XGBoost 基线在测试集上取得 detection `AUC = 0.8199`，说明原始触觉信号中首先存在稳定的结节存在信息。进一步地，XGBoost 在大小任务上取得 `Top-1 = 0.6701`、`Top-2 = 0.8023` 和 `MAE = 0.1472 cm`，表明大小信息并不是偶然噪声，而是能够被结构化特征稳定反演。对于更困难的深度任务，XGBoost 仍取得 `balanced accuracy = 0.5138`，显著高于多数类基线 `0.3333`。因此，结构化 baseline 首先证明的不是“神经网络更强”，而是：**通过原始触觉信号进行结节探测、大小反演和粗深度辨别是可能的。**

### 3.3 SHAP 显示三个问题依赖的物理特征并不相同
SHAP 分析进一步表明，检测任务主要依赖明显的幅值和集中度差异；大小任务主要依赖峰值、能量和中心-边缘对比等强特征；而深度任务的主导特征则更偏向 spread、shape contrast、deformation position 和 temporal phase。这个结果说明，深度不是“再用同一组强度特征做一次分类”，而是一个机制上更复杂的问题，也解释了为什么其建模难度明显高于 detection 和 size。

### 3.4 raw-input 神经网络在 detection 上首先超过了结构化基线
在独立测试集 `3.CSV` 上，阶段一 raw-input 检测器取得 `AUC = 0.8383`、`AP = 0.5357` 和 `F1 = 0.6216`，整体优于 XGBoost 的 `0.8199`、`0.5063` 和 `0.6185`。这一结果说明，在首要任务 detection 上，原始时空张量本身已经足以支撑稳定判断，而且其表现可以超过手工结构化特征路线。

### 3.5 raw-input 神经网络进一步在 size 任务上超过了 XGBoost
优化后的 raw-input size-only router v2 在真实阳性窗口上达到 `Top-1 = 0.7177`、`Top-2 = 0.8195` 和 `MAE = 0.1242 cm`，全面优于 XGBoost 的 `0.6701`、`0.8023` 和 `0.1472 cm`。这一步非常关键，因为它表明：在不直接输入结构化物理特征的情况下，raw-input 神经网络不仅能学到“有没有结节”，还能够进一步稳定反演强主效应的大小信息。

### 3.6 深度只有在 size-aware 与 route-aware 组织下才真正变得可学
最初的普通 raw-input 深度头在严格重复划分协议下退化到多数类附近，说明深度任务不能被当作一个普通共享头直接处理。当深度改写为 size-routed experts 后，ground-truth route 的平衡准确率首先提高到 `0.5238`，略高于 XGBoost 的 `0.5138`。在进一步引入 route-aware 训练后，当前 route-aware depth v2 的 `GT-route balanced accuracy = 0.6066`，而更关键的 `predicted-route balanced accuracy = 0.5240`，也首次稳定超过 XGBoost 的 `0.5138`。这个结果说明：深度并不是不可学，而是**必须在大小条件下被组织起来**。

### 3.7 可解释性结果表明网络确实隐式编码了关键物理结构
在当前 route-aware depth v2 模型上，latent probe 的平均测试 `R^2 = 0.2712`，高于 size-only probe 的 `0.2356`。这说明模型内部并不只是保留了大小标签，而是编码了部分与深度相关的更细结构。进一步地，可恢复性较高的概念包括 distribution complexity、shape contrast、amplitude response 和 rise-time-to-peak，表明网络学到的并不是单一维度的捷径。

Integrated Gradients 进一步显示，在每个深度类别各选取 40 个正确预测测试窗口后，`shallow / middle / deep` 三类对应的平均空间归因区域并不完全相同，模型在空间上已经形成了与类别相关的关注结构。与此同时，hard-pair analysis 表明，在构造的困难样本对中，更深样本在约 `75%` 的配对中获得更高的 `p_deep`，说明模型并非单纯依赖峰值幅值完成判断。

不过，phase occlusion 也给出了一个同样重要的反向信息：遮挡 `peak neighborhood` 时，真实类别概率平均下降达到 `0.2303`，明显高于 `loading early (0.0662)` 与 `release (0.0446)`。这意味着当前模型虽然已经自动编码了部分与深度相关的物理结构，但其时间策略仍偏向峰值邻域，对完整时序机制的利用仍不充分。

### 3.8 从 XGBoost 到 raw-input 神经网络，本文形成了一条完整证据链
将上述结果连起来看，本文的证据链是清楚的：
1. 文献、机制分析和结构化特征先提出候选物理线索；
2. XGBoost 证明 detection、size 和 coarse depth 在结构化特征空间中确实可学；
3. raw-input 神经网络进一步证明原始时空张量本身可以学习这些信息，并在关键任务上超过 XGBoost；
4. 解释性分析最后说明，神经网络内部确实自动编码并利用了与前述结构化特征一致的部分关键结构。

这条证据链回答的不是“哪个模型名义上更先进”，而是“原始触觉时空信号是否真的包含这些信息，以及模型是否真的学到了这些信息”。

### 3.9 部署增强模型与实时系统验证了工程可行性
尽管 pure raw route-aware 模型已经在 predicted-route 深度任务上超过 XGBoost，但真实部署仍然受 `size -> depth` 路由误差影响。为此，本文进一步构建统一层级反演器，在联合原始时空输入与结构化特征后，将 predicted-route 深度平衡准确率进一步提升至 `0.5337`。最终，该体系被部署到实时 GUI 中，形成 detection-first 的术中触觉导航原型。系统持续显示 detection 结果，并仅在高置信条件下释放 size 和 depth 输出，从而更符合实际使用场景。

## 4 讨论
本文最重要的特点，并不是简单提出了一个新的神经网络，而是把术中肺结节触觉定位问题重新组织成了一条更自然的研究路径。对于这一问题，最直接、最临床相关的需求是结节探测；大小信息是在探测成立后自然浮现出的强主效应；而深度则是在实验设计中真实存在、但更弱且更耦合的变量，因此只能被更谨慎地定义为 coarse depth discrimination。这样的任务组织方式，使得模型设计、结果解释和工程部署能够保持一致。

从方法论上看，本文采用的“机制分析与结构化 baseline 先行，再进入 raw-input 深度学习，最后用可解释性闭环验证”的范式，并非多余步骤，而是当前这类 biomedical engineering 问题中更稳健的做法。一方面，它避免了把全部结论都压在一个黑箱模型上；另一方面，它允许我们先回答“信息是否存在”，再回答“原始输入是否足以自动学习这些信息”。这一点对样本规模有限、问题带有明显物理耦合和解释性要求的触觉场景尤其重要。

关于深度问题，本文的结果支持一个相对克制但清晰的结论：深度不是不可学，但也远未达到“连续深度反演”这一更强主张。XGBoost 与 raw-input 路线都表明深度信息客观存在，而 route-aware raw model 在 predicted-route 条件下也已超过 XGBoost；然而，phase occlusion 同时说明模型仍明显依赖 peak neighborhood，这意味着当前网络尚未充分利用 loading / release 等更完整的时序信息。因此，本文关于深度的定位应当是**探索性的粗粒度辨别**，而不是已经解决的精确反演。

本文也存在几个需要明确界定的边界。第一，当前评估协议本质上是基于重复实验的 `repeat-held-out` 验证，而不是更严格的 `leave-condition-out` 泛化评估，因此结果应主要解释为在固定物理设计矩阵下的稳定验证。第二，当前 explainability 结果已经能支持“网络确实学到了部分物理结构”这一结论，但并不能说明网络已经完整复现了前述机制分析中涉及的全部概念。第三，当前系统原型已具备工程可用性，但距离真实临床环境下的广泛验证仍有明显距离。

尽管如此，本文仍然给出了一个相对扎实的起点：柔性触觉时空数据不仅能够支持结节探测，还可以进一步支持大小反演和粗深度辨别；而且这种能力既可以通过结构化特征被验证，也可以由 raw-input 神经网络直接学到，并通过解释性分析得到反向支撑。这为后续面向更复杂组织条件、更严格泛化协议和临床环境的持续研究提供了清晰方向。

## 5 结论
本文围绕术中肺结节触觉缺失问题，提出并验证了一条更符合问题自然展开方式的研究路线：从可靠探测出发，逐步发现大小信息，再谨慎进入深度问题；从文献和机制分析出发，先用结构化特征与 XGBoost 建立可学性证据，再进一步使用 raw-input 层级神经网络完成自动学习与解释性验证。结果表明，柔性触觉时空数据能够首先支持稳定的结节探测，并进一步支持大小反演和粗深度辨别；同时，当前 raw-input 神经网络在 detection、size 和 predicted-route depth 等关键任务上已达到甚至超过结构化基线。整体上，本文更接近于提出一个“检测优先、反演增强、解释性验证、部署增强”的术中触觉定位框架，而不是宣称已经完成一个临床终态系统。这种渐进式证据链，是本文最核心的贡献。
