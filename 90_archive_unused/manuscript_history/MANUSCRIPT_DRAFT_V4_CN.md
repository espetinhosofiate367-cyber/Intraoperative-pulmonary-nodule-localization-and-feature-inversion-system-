# 基于柔性阵列触觉传感与层级时空神经网络的术中肺结节定位与特征反演系统

## 摘要
随着低剂量计算机断层扫描在肺癌早期筛查中的广泛应用，临床上对微小肺结节及磨玻璃结节的检出率持续升高。对于此类高度疑似早期病灶，电视辅助胸腔镜手术和机器人辅助手术虽然显著降低了创伤，但也使术者失去了开放手术中最重要的直接触觉反馈能力，从而增加了深部、小尺寸和隐匿性结节在术中定位中的困难。围绕这一临床痛点，本文构建了一套基于柔性阵列触觉传感与层级时空神经网络的术中肺结节定位与特征反演系统。系统以时序应力图窗口为输入，依次执行结节检测、结节大小反演以及大小条件下的深度粗分类，并将反演结果实时加载到术中可视化界面中。针对深度信息弱、与结节大小高度耦合且易受路由误差影响的问题，本文首先通过非深度学习机制分析和 XGBoost 基线识别出深度相关的扩散、形态、分布复杂度和时序阶段特征，再据此设计 size-routed 深度网络与统一层级反演器。实验结果表明，阶段一检测模型在独立测试集上的 AUC 为 0.8383，高于 XGBoost 的 0.8199；统一层级反演器在预测路由条件下的深度粗分类平衡准确率达到 0.5337，超过 XGBoost 的 0.5138；原始输入神经网络在 size-aware 条件下能够学习粗深度信息，并通过 latent probe、hard-pair analysis 与 phase occlusion 形成可解释性闭环。研究表明，原始时空触觉数据中确实包含可学习的深度信息，但其稳定利用依赖于层级任务组织、大小感知路由以及对真实部署误差传播的显式优化。

**关键词**：肺结节定位；柔性阵列传感器；术中导航；触觉反演；层级神经网络；可解释人工智能

## 1 引言
随着低剂量计算机断层扫描在肺癌早期筛查中的普及，临床上对无症状微小肺结节及肺部磨玻璃结节的检出率呈持续上升趋势。对于此类高度疑似早期恶性病灶，电视辅助胸腔镜手术和机器人辅助手术凭借创伤小、恢复快和住院周期短等优势，已逐渐成为临床主流干预方式。然而，微创技术在降低创伤的同时，也引入了一个长期存在而尚未被彻底解决的术中定位难题：外科医生在依赖细长器械操作时，几乎完全丧失了开放手术中可直接依赖的触觉反馈能力。对于距离肺表面较深、尺寸较小或缺乏明显牵拉形变的结节，这种触觉缺失会显著增加漏检、扩大切除范围甚至中转开胸的风险。

在现有临床实践中，为弥补触觉信息不足，术者通常借助术前 CT 引导定位、染料或金属钩标记、术中超声以及有限度的数字触诊等方式辅助识别病灶。然而，这些方案均存在明确局限。侵入式术前定位伴随气胸、出血、标记移位等并发症风险；术中超声对探头贴合、空气干扰控制和操作者经验具有较高依赖；数字触诊则高度依赖个人经验，缺乏客观量化依据。归根结底，当前临床尚缺少一种既能保留微创优势、又能在术中稳定恢复“触觉信息”的实时定量方案。

柔性阵列触觉传感器为恢复术中触觉提供了一条有前景的技术路线。与传统单点力传感器不同，柔性阵列能够在按压过程中连续采集二维表面应力分布，并进一步形成具有时间演化信息的时空应力张量。当高刚度结节埋藏于软组织内部时，结节与周围组织之间的机械不匹配会改变表面应力场的峰值、扩散范围、空间形态以及随时间的演化方式。理论上，这些表面响应中包含病灶是否存在、病灶大小以及埋藏深度等多层信息。

然而，将表面触觉矩阵进一步反演为内部结节参数，本质上是一个病态逆问题。一个浅层但较小的结节，可能在某些按压阶段产生与一个深层但较大的结节相似的表面峰值；若仅依赖单帧应力图或简单强度阈值，模型容易将深度效应误判为大小效应，或者反过来将大小差异错误解释为深度差异。对于医疗场景而言，仅仅得到“可用的分类结果”仍然不够，模型还必须具备一定的物理可解释性，才能回答“为什么做出这样的深度判断”，并为临床使用建立可信基础。

围绕这一问题，本文采用了“先机制分析，再层级建模，再解释验证”的研究路径。首先，我们在 7 个结节尺寸、6 个埋藏深度和 3 次重复实验构成的设计矩阵下，对结节存在、大小和深度对表面应力响应的影响进行了系统分析。结果表明，结节大小是强主效应，主要影响峰值强度、整体能量和中心集中度；深度则是较弱但条件依赖的效应，主要体现为扩散范围、分布复杂度、形变位置以及按压阶段相关的时序变化。随后，我们建立 XGBoost 基线，验证这些结构化物理特征中已包含可学习的大小与深度信息。最后，在此基础上设计原始输入神经网络，并逐步证明：若不显式考虑 size-depth 耦合，网络会在深度任务上退化到多数类附近；只有将深度建模为大小条件下的细粒度判别问题，原始输入神经网络才能稳定学习粗深度信息。系统总体框架见图1，最终训练与推理流程见图2。

需要特别说明的是，本文引入 XGBoost 并不是为了与神经网络形成简单的“谁分数更高”的替代关系，而是承担三个不同层次的研究角色。第一，XGBoost 作为结构化物理特征基线，用于验证“数据中是否真实存在可学习的大小和深度信息”；如果这一层都无法成立，则后续深度学习结果缺乏物理依据。第二，XGBoost 提供一个中间解释锚点，使我们能够在不依赖黑箱表示的前提下识别出 deformation-position、shape-contrast 和 spread-related 等关键特征家族，从而为神经网络结构设计提供约束。第三，神经网络的目标并不是重复结构化基线，而是进一步回答 XGBoost 无法直接解决的问题，即如何从原始时空触觉窗口中自动恢复这些信息、如何在真实 size routing 误差存在时保持 depth 判别鲁棒性，以及如何将模型部署到实时术中界面。因此，本文的方法链条不是“先做一个传统模型，再换成更复杂模型”，而是“先用结构化基线证明和解释信息，再用神经网络从原始输入中学习并超越这一结构化参照”。

从方法组织上，本文并未将“检测、大小、深度”视作三个彼此独立的平级输出，而是将其组织为一个逐级细化的层级推理系统。该思想与层级学习框架中“先粗后细”的逻辑相一致，但并未照搬病理全切片图像中的 bag-instance 多实例学习结构，而是结合触觉时序任务特点，将层级关系具体落实为：首先完成结节检测，其次进行大小反演，最后在大小条件下执行深度粗分类，并通过 route consistency 约束真实部署中的误差传播。这样做的核心目的，是避免网络将“大而深”简单识别为“小而浅”，并使深度判别真正建立在扩散、形态和时序机制之上。

本文的主要贡献如下：

1. 构建了面向术中触觉定位的结构化时空数据体系。在 7 个结节尺寸、6 个埋藏深度和 3 次重复实验条件下，获得 126 组具有真实时序特征的逐帧应力分布序列，并完成了与检测、大小反演和深度粗分类相匹配的窗口级标签协议设计。

2. 提出了一套层级式术中肺结节反演系统。系统由阶段一检测器、阶段二大小反演器和阶段三大小感知深度分类器组成，并进一步构建统一层级反演器，直接优化预测路由条件下的深度性能。

3. 证明了原始输入神经网络可以学习粗深度信息，但前提是显式尊重深度的大小依赖结构。在 ground-truth size route 条件下，原始输入的 size-routed 深度网络已可达到与 XGBoost 基线相当甚至略优的性能；在进一步考虑真实预测路由后，统一层级反演器将 predicted-route 深度性能稳定推过了 XGBoost。

4. 建立了从机制分析、结构化基线到神经网络内部表征的可解释性闭环。通过 XGBoost 特征贡献分析、latent linear probes、hard-pair analysis 与 phase occlusion，本文不仅回答了“模型能否判别深度”，也回答了“模型主要依赖哪些深度相关特征以及仍存在哪些偏差”。

## 2 材料与方法
### 2.1 数据采集与实验设计
本研究使用柔性阵列触觉传感系统对离体猪肺模型进行动态按压实验。每一帧包含 96 个传感通道，经重排后形成 12 × 8 的二维应力分布图。实验设计矩阵由 7 个结节尺寸（0.25、0.5、0.75、1.0、1.25、1.5、1.75 cm）和 6 个埋藏深度（0.5、1.0、1.5、2.0、2.5、3.0 cm）组成，共 42 个物理条件，每个条件进行 3 次独立重复测试，最终形成 126 组带有完整时序特征的逐帧应力数据序列。

为了适配网络输入，本文采用长度为 10 帧的滑动窗口作为基本样本单位，步长为 2。检测任务采用窗口中心帧是否位于人工标注阳性片段内作为标签；大小和深度标签由该记录的实验条件继承而来。检测任务使用 `1.CSV + 2.CSV` 作为开发集、`3.CSV` 作为独立测试集；大小和深度反演任务则采用更严格的重复划分协议，即 `1.CSV` 训练、`2.CSV` 验证、`3.CSV` 测试，并仅在阳性窗口上训练。

### 2.2 非深度学习机制分析与结构化基线
在神经网络训练之前，本文首先对帧级和窗口级应力特征进行了系统分析。统计特征包括峰值强度、积分响应、中心-边缘对比、热点面积、热点半径、二阶矩扩散、空间熵、质心漂移、峰值到达时间、衰减特征以及若干窗口级时序统计量。分析结果表明，结节大小是强主效应，而深度主要影响扩散、分布复杂度、形变位置以及时序阶段响应。

在此基础上，本文建立了 XGBoost 结构化基线，用于完成检测、大小分类/回归和深度粗分类任务。该基线的作用有两点：一是验证数据中确实存在可学习的大小与深度信息；二是为后续神经网络的可解释性提供结构化参照。实验结果显示，XGBoost 在检测任务上略弱于最优神经网络，但在深度粗分类任务上可达到 0.5138 的平衡准确率，说明深度信息虽然较弱，但并非不可学习。
为了使这一结构化基线的解释结果更符合生物医学工程论文的阅读习惯，本文进一步采用了面向特征家族、类别级概念模式、代表性局部样本和偏依赖趋势的多面板可视化组织方式，而不是仅报告单一条形重要性图。这种图表设计使得“模型整体依赖什么”“不同深度类别依赖什么”以及“代表性样本为何被判为 shallow / middle / deep”能够在同一图中得到较完整呈现。
换言之，XGBoost 在本文中承担的是“结构化机制基线”和“解释性桥梁”的角色，而非最终部署模型。我们之所以在建立 XGBoost 之后仍然需要神经网络，是因为 XGBoost 依赖人工设计特征，无法直接处理原始时空触觉窗口，也无法自然表达真实部署中 `size -> depth` 的误差传播与路由鲁棒性问题。神经网络的任务因此被明确限定为：在不直接喂入手工深度特征的前提下，从原始输入中恢复这些可学习信息，并通过层级建模把这种能力转化为可部署的系统性能。

### 2.3 层级时空神经网络系统
#### 2.3.1 阶段一：结节存在检测
检测阶段采用双流时空网络，对原始窗口和相邻帧差分窗口并行编码。空间维使用轻量卷积编码器提取，时间维通过 MS-TCN 聚合，最终输出结节存在概率。该设计的目标是先稳定回答“当前窗口下方是否存在结节”，从而使后续反演仅在高置信窗口上执行。

#### 2.3.2 阶段二：大小反演
大小反演阶段仅在阳性窗口上运行。网络直接从原始时序应力窗口学习 7 类大小分类和连续大小回归，并作为后续深度判别的路由条件。由于前期分析已表明大小是强主效应，本文为大小任务单独训练专用 router，以减少共享多任务头带来的误差传播。

#### 2.3.3 阶段三：大小感知深度粗分类
针对深度任务，本文并未采用单一共享的深度分类头，而是使用 size-routed depth experts。该设计的出发点在于：深度效应并不是全局一致的独立信号，而是在不同大小条件下呈现出不同的应力扩散和时序模式。因此，网络首先依据大小信息选择对应的深度专家，再进行 `shallow / middle / deep` 的粗分类判别。

#### 2.3.4 统一层级反演器
为了进一步提高真实部署条件下的性能，本文构建了统一层级反演器。该模型联合输入原始幅值序列、归一化形态序列以及筛选后的结构化物理特征，并同时输出大小分类、大小有序预测、大小回归和 routed depth。与早期的“先 size 再 depth”的松耦合串联方式不同，统一层级反演器在训练中显式加入了 ground-truth route、hard predicted route、soft predicted route、top2 route 以及 route consistency 目标，从而直接优化真实部署链路中的 predicted-route depth 表现。统一系统与其部署接口可分别参见图1和图2。

### 2.4 可解释性分析框架
为了回答“网络究竟学到了哪些深度相关信息”，本文从三个层面构建解释性框架：

1. **机制先验层**：利用非深度学习统计分析明确候选深度敏感特征，包括扩散、形态、位置和阶段响应。

2. **结构化基线层**：通过 XGBoost 特征贡献分析验证哪些特征家族主导了深度判别。

3. **神经网络内部表征层**：对训练后的深度网络执行 latent linear probes、hard-pair analysis 和 phase occlusion，以判断其内部表征是否真正编码了超出大小本身的深度相关结构。

## 3 结果
### 3.1 检测结果表明原始输入时空神经网络在结节存在判别上优于结构化基线
在独立测试集 `3.CSV` 上，阶段一检测模型（raw+delta）取得 AUC 0.8383、AP 0.5357 和 F1 0.6216。相同协议下，XGBoost 检测基线的 AUC、AP 和 F1 分别为 0.8199、0.5063 和 0.6185。该结果表明，在结节存在检测这一以局部时空模式为主的任务上，原始输入神经网络已经优于结构化树模型基线，说明时空表征学习能够更有效地利用动态应力分布中的判别信息。

### 3.2 大小反演结果说明结节大小是强主效应，但不同架构在“独立 size 最优”和“链路整体最优”之间存在权衡
在测试集真实阳性窗口上，XGBoost 的大小分类/回归性能分别为 top-1 0.6701、top-2 0.8023 和 MAE 0.1472 cm。专用 raw-input size-only router 的 top-1 和 top-2 分别达到 0.6600 和 0.8405，但 MAE 为 0.2907 cm；统一层级反演器的 top-1、top-2 和 MAE 分别为 0.6180、0.7859 和 0.1565 cm。由此可见，若仅以独立的大小回归误差为目标，XGBoost 仍保持最优 MAE；而若考虑后续深度路由的稳定性，统一层级反演器虽然牺牲了部分 standalone size 指标，却为下游 depth 任务提供了更有利的整体条件。

### 3.3 结构化基线证明深度并非不可学，而是一个明显弱于大小且依赖物理结构组织的信号
XGBoost 在深度粗分类任务上的准确率和平衡准确率分别达到 0.5151 和 0.5138，显著高于多数类基线的 0.3333。进一步的特征贡献分析显示，深度判别主要依赖 deformation-position、shape-contrast 和 spread-related 特征家族，而纯 amplitude family 的贡献相对较小。这一结果明确说明，深度信息在数据中真实存在，但其主要表现形式并非简单峰值衰减，而是扩散、形态和位置的联合变化。对应的机制分析与 XGBoost 解释汇总见图11。
值得强调的是，XGBoost 的解释结果并非只给出“哪些特征重要”，还通过类别级概念热图和代表性触觉图进一步显示：浅层、中层和深层样本在形变位置、热点扩散和分布复杂度上的依赖模式并不相同。这一点为后续神经网络的 size-aware depth 设计提供了直接依据。

### 3.4 若不显式处理 size-depth 耦合，原始输入深度网络会退化到多数类附近
最初的 raw-input end-to-end 深度模型在严格重复划分协议下表现接近多数类基线，其测试平衡准确率约为 0.33。该结果的意义并不在于“原始输入不能做深度”，而在于它证明了一个更关键的事实：当大小是强主效应、深度是弱主效应时，若仍采用普通共享 depth head，网络会优先吸收强度和大小相关线索，最终导致深度判别被淹没在更强的因素中。

### 3.5 引入 size-routed depth experts 后，原始输入神经网络可以恢复 coarse depth 信息
当单一 depth head 被替换为 size-routed depth experts 后，原始输入深度网络的性能显著改善。在 ground-truth size route 条件下，该模型的测试平衡准确率达到 0.5238，已经与 XGBoost 的 0.5138 基本相当并略有超过。这一结果说明，粗深度信息确实可以直接从原始时空触觉窗口中学习得到，但前提是模型必须尊重 depth 的 size-dependent 结构，而不能将深度视作脱离大小而独立存在的单头分类任务。

### 3.6 latent probes 表明 size-routed 深度网络内部编码了超出 size 本身的深度相关结构
为了验证 raw-input size-routed 深度网络内部是否真的学习到了具有物理意义的深度敏感信息，本文冻结训练后的网络 trunk，并在其潜在表示上训练线性 ridge probes 去恢复前期机制分析中识别出的结构化特征。结果显示，latent probe 的平均测试 `R^2` 为 0.3657，而仅以 size identity 为输入的 probe 为 0.2356，前者明显更高。尤其在 distribution complexity、temporal phase 和 deformation position 等特征家族上，latent representation 的可恢复性显著优于 size-only probe。该结果说明，网络内部并不是简单“记住了大小”，而是已经编码了一部分真正与深度相关的结构信息。

### 3.7 hard-pair analysis 说明模型并非只依赖峰值幅值，但对困难样本的分离仍有限
为了进一步检验模型是否会把“大而深”误判成“小而浅”，本文构造了若干 hard-pair 样本对。在这些样本中，浅层小结节和深层大结节在部分峰值强度指标上具有相似性，但其扩散、复杂度和时序结构不同。结果显示，更深样本在约 75% 的 hard-pair 对中获得更高的 `p_deep`，说明模型并非单纯依赖峰值幅值进行判断。然而，由于这些样本对在部分 amplitude 量上仍存在残余差异，因此当前证据更适合写成“中等强度支持”，尚不足以宣称模型已经完全解决了“大而深/小而浅”混淆问题。

### 3.8 phase occlusion 表明当前网络仍偏依赖峰值邻域，而尚未充分利用 loading/release 阶段的深度线索
phase occlusion 分析进一步揭示了网络的时间策略。遮挡 `peak neighborhood` 后，真实深度类别概率的平均下降幅度最大；而遮挡 `loading early` 和 `release` 阶段造成的下降相对较小。这一点与前面非深度学习机制分析并不完全一致，因为机制分析提示早期加载与释放阶段更可能携带深度相关信息。由此可见，当前 raw-input 深度网络已经学到了一部分正确的深度结构，但其时间注意力仍然过度集中在峰值邻域，这也是后续模型迭代的重要方向。

### 3.9 真实部署链路中的主要误差来自 size routing，而非 depth trunk 本身
虽然 raw-input size-routed 深度模型在 oracle route 条件下性能良好，但真实部署必须依赖预测 size route。实验表明，若使用早期共享多任务 size head 进行路由，Stage3 正窗上的 size top-1 仅约 0.5226，从而使 predicted-route 深度平衡准确率下降至 0.4822。进一步分析发现，当 size route 正确时，depth 的平衡准确率可达 0.5557；而当 route 错误时，该指标降至 0.3204。该结果清楚说明，当前链路中的主要误差传播源并不是 depth trunk 本身，而是上游 size router 的错误将样本引向了错误的深度专家。

### 3.10 统一层级反演器在真实 predicted-route 条件下首次稳定超过 XGBoost
针对上述瓶颈，本文进一步构建了统一层级反演器，将大小判别、路由策略与深度专家联合组织在同一训练图中，并显式优化 ground-truth route、hard route、soft route、top2 route 以及 route consistency。实验结果显示，该模型在测试集上的 predicted-route depth hard balanced accuracy 达到 0.5337，不仅高于旧 predicted-route 链路的 0.4822，也超过了 XGBoost 的 0.5138。同时，该模型在 ground-truth route 条件下的 depth 平衡准确率达到 0.5407。该结果表明，若论文评价目标从“单点 size 指标最优”转向“真实部署链路中的深度判别鲁棒性”，统一层级反演器是更合理的最终系统形式。

### 3.11 统一层级反演器的解释结果表明，最终主文模型同时依赖原始时空线索与结构化物理线索
为了使最终主文模型与解释分析保持一致，本文进一步对统一层级反演器执行 latent probe、分支消融、hard-pair 与 phase occlusion 分析。冻结模型后，在其 `raw trunk` 与 `hybrid latent` 上训练线性 ridge probes 去恢复训练中使用的结构化深度相关特征。结果显示，`hybrid latent` 的平均测试 `R^2` 为 0.5512，`raw trunk` 为 0.4226，而仅以 size identity 作为输入的 `size-only probe` 为 0.2985。该结果说明，统一架构的内部表示并非仅仅编码了大小本身，而是确实保留并整合了与深度相关的空间扩散、分布复杂度、形变位置和时序变化信息。

分支消融结果进一步说明了最终模型依赖的信息来源。去除 `raw` 分支后，depth hard-route 的平衡准确率由 0.5337 降至 0.4165；去除 `shape` 分支后，该指标降至 0.3811；去除 `delta` 分支后下降至 0.4705；去除 `tabular` 分支后下降至 0.3742。与此同时，size top-1 也分别从 0.6180 降至 0.5488、0.1689、0.6023 和 0.3935。上述结果表明，统一层级反演器并非依赖单一峰值强度或某一个局部分支做出判断，而是通过原始幅值、归一化形态、时序变化和结构化物理特征的联合表征来实现大小与深度反演，其中 `shape` 与 `tabular` 分支对最终性能尤其关键。

在最终模型的 hard-pair 分析中，构造得到的 10 对“大而深 / 小而浅”样本中，较深样本在 80% 的配对中获得更高的 `p_deep`。这说明最终模型已经具备一定程度的困难样本分离能力，并非简单依赖单点峰值幅值完成 depth 判别。不过，由于这些样本对在 `raw_max` 上仍存在较大的残余差异（均值约 50.55），因此该证据当前更适合表述为“中等到较强支持”，而非“已经完全解决大深/小浅混淆”。

phase occlusion 结果则表明，统一层级反演器在时间维上的主要依赖仍集中在 `peak neighborhood`。遮挡该阶段后，真实深度类别概率的平均下降幅度最大（mean drop 0.0802）；相比之下，`loading early` 和 `release` 的影响较弱。该结果说明，虽然最终系统在部署性能上已经优于结构化基线，但它尚未完全对齐前面机制分析中提示的“loading/release 对深度更敏感”的物理规律，因此 phase-aware 建模仍是后续优化的重要方向。统一层级反演器的解释结果汇总见图12。

## 4 讨论
本文的主要结论可以概括为三个层次。首先，从机制分析和 XGBoost 基线可以确认，深度并非不可学，而是一个显著弱于大小、且在不同大小条件下表现不同的信号。其次，从 raw-input 神经网络的对比实验可知，若网络结构不显式处理 size-depth 耦合关系，深度学习会退化到多数类附近；而一旦使用 size-routed 专家结构，原始输入即可学出可用的 coarse depth。最后，从真实部署链路的 predicted-route 评估可知，系统的关键不只是提升 depth trunk 本身，而是减少 size route 误差向 depth 的传导，这也是统一层级反演器优于松耦合串联方案的根本原因。

本研究还揭示了一个对临床部署尤其重要的事实：单点的“最好 size 指标”并不必然等价于“最好系统性能”。在我们的实验中，XGBoost 仍保持最优的 size MAE，但统一层级反演器在真实 predicted-route depth 上获得了更好的结果。这意味着，对于术中系统而言，更重要的问题并不是孤立地优化某一个中间任务，而是让整条推理链在真实误差传播条件下保持鲁棒。

从叙事逻辑上看，XGBoost 与神经网络之间并不是竞争关系，而是递进关系。XGBoost 首先回答“哪些结构化触觉量与深度相关、这些信息是否足以支撑粗分类”；神经网络随后回答“在不直接输入这些手工特征的情况下，原始时空触觉数据能否自行形成类似的内部表征，并在真实部署链路中保持更好的鲁棒性”。因此，当神经网络在 detection 上超过 XGBoost、在 GT-route depth 上达到并略超 XGBoost、并最终在 predicted-route depth 上稳定超过 XGBoost 时，这条证据链实际上是在说明：神经网络并不是绕开了前面的结构化认识，而是在前述机制约束之上，把结构化线索重新吸收到原始输入模型中，并进一步把它们组织成更适合实时系统部署的层级推理过程。

需要指出的是，本文当前结果仍存在几个边界。其一，统一层级反演器虽然在 predicted-route depth 上取得了最佳结果，但其单独的 size 指标尚未在所有维度上全面超过 XGBoost；其二，hard-pair 与 phase occlusion 分析提示模型已经具备部分深度可解释性，但对 loading/release 阶段的利用仍不足；其三，本文的深度任务目前仍定位于 coarse depth classification，而非更细粒度的连续深度反演。因此，在后续工作中，需要进一步强化 phase-aware 建模、引入更针对路由噪声的训练策略，并推动由粗深度向更细粒度深度推断的扩展。

## 5 结论
本文围绕术中肺结节定位中的触觉缺失问题，构建了一套基于柔性阵列触觉传感与层级时空神经网络的定位与特征反演系统。研究结果表明：结节大小是表面应力响应的强主效应，而深度则是较弱但可学习的条件依赖效应；粗深度信息并非无法从原始触觉数据中恢复，而是必须通过大小感知路由和层级结构设计来显式建模；在真实部署链路中，统一层级反演器已将 predicted-route depth 性能稳定提升至超过 XGBoost 基线的水平。更重要的是，本文通过机制分析、结构化基线和神经网络解释验证形成了较完整的证据链，使“模型为什么能判别深度”这一问题获得了可检验而非纯黑箱的回答。该研究为柔性触觉传感在术中肺结节定位中的应用提供了一个可部署、可解释、可进一步迭代的技术基础。
