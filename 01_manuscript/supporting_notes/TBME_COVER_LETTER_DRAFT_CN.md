# TBME Cover Letter 草稿（中文工作版）

尊敬的编辑：

我们谨提交题为《基于触觉感知与深度学习的术中肺结节定位系统：一项前沿性探索研究》的稿件，申请作为 `IEEE Transactions on Biomedical Engineering` 的原创研究论文审阅。

本研究面向微创肺结节手术中触觉缺失导致的术中定位困难，提出并验证了一条由柔性触觉感知、力学启发机制分析、原始时空深度学习、可解释性验证与部署增强共同构成的技术路线。与仅比较模型分数的工作不同，本文首先通过结构化特征分析与 XGBoost 基线证明柔性触觉数据中存在可学习的探测、大小及粗深度信息；随后构建不直接输入结构化物理特征的 raw-input 层级神经网络，证明原始时空张量本身已足以支持稳定探测、较强大小学习及大小条件下的粗深度辨别；最后再通过解释性分析验证网络内部确实自动编码了部分与大小和深度相关的物理结构，并以统一层级反演器进一步提升真实 predicted-route 条件下的系统性能。

我们认为该稿件与 TBME 关注的 biomedical instrumentation, biosensing, biomechanics, signal processing, and diagnostic system integration 高度契合。本文的贡献不在于宣称已经完成临床定型系统，而在于以严谨的数据协议、清晰的机制链和可解释的系统组织，展示了术中触觉定位问题的一条可验证、可工程化的前沿技术路线。

此稿件未在其他期刊投稿，也未曾发表。感谢您和审稿专家的审阅。
