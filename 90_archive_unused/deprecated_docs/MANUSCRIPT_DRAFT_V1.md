# Manuscript Draft v1

## Title
Flexible Array Tactile Sensing for Intraoperative Lung Nodule Localization: Detection, Size Inversion, and Interpretable Coarse Depth Classification From Spatiotemporal Stress Maps

## Abstract
Intraoperative localization of pulmonary nodules remains difficult when lesions are small, deeply embedded, or weakly visible on the pleural surface. This study investigates whether a flexible array tactile sensor can support lung nodule localization by converting pressing-induced stress distributions into machine-learning estimates of nodule presence, size, and depth. We designed a phantom-based dataset with 42 size-depth conditions generated by 7 nodule diameters (0.25-1.75 cm) and 6 depths (0.5-3.0 cm), with three repeated experiments per condition. Each recording consists of frame-wise 12 x 8 stress maps acquired during pressing, with manually labeled nodule-positive segments. We first established non-deep-learning baselines and mechanism analyses to identify which stress characteristics vary with size and depth. Size showed a strong and stable effect on peak amplitude, integrated response, and center-border contrast, whereas depth exhibited weaker but measurable effects concentrated in spread, distribution complexity, deformation position, and phase-dependent response. An XGBoost baseline using handcrafted physical descriptors achieved a coarse depth balanced accuracy of 0.514, indicating that depth information is present in the data but is structurally weaker than size. For neural models, a naive raw-input depth classifier collapsed to majority-class behavior unless size-aware routing was imposed. A raw-input size-routed depth network using amplitude, normalized shape, and frame-difference streams reached a test balanced accuracy of 0.512 under ground-truth size routing, matching the XGBoost depth baseline. Latent linear probes further showed that the trained model encoded depth-relevant descriptors beyond size identity, especially distribution complexity, temporal phase, and deformation-position cues. However, phase occlusion revealed that the current model still relied mainly on the peak neighborhood rather than fully exploiting the loading and release phases suggested by the non-deep-learning mechanism analysis. In deployment-mode evaluation, depth performance was strongly limited by upstream size-routing errors. A dedicated raw-input size router increased size top-1 accuracy to 0.660 on positive windows and improved predicted-route depth balanced accuracy from 0.441 to 0.482. These results support a three-stage system design: detection first, size estimation second, and size-aware depth classification third. The findings also show that interpretable depth prediction is feasible from raw tactile input, but only when the network architecture explicitly respects the size-dependent nature of the depth cue.

## 1. Introduction
Intraoperative identification of lung nodules is clinically important because failure to localize the lesion can enlarge the resection range, prolong the operation, or require additional localization procedures. The problem becomes more difficult for small nodules and for lesions that are not visible on the lung surface. Tactile sensing is a natural candidate because subsurface abnormalities alter local compliance and therefore change the surface stress pattern during palpation.

Flexible array sensors are attractive for this setting because they provide a dense spatiotemporal stress map rather than a single-point force measurement. In principle, if a nodule is embedded below the pleural surface, pressing the tissue should produce a more concentrated stress response over the lesion area. However, two questions remain open. First, can raw frame-wise stress maps support stable detection and size estimation? Second, can the same measurements provide interpretable depth information rather than only a coarse presence signal?

The present study addresses these questions with a staged strategy. We first quantify what can be learned without deep learning, using descriptive statistics, mechanism analysis, and XGBoost baselines on handcrafted physical features. We then use these findings to constrain the design of neural networks. The central technical issue is that size is a strong effect, whereas depth is a weaker, conditional effect. If the network ignores this imbalance, large deep nodules may be confused with smaller shallow nodules because both can produce strong responses. Therefore, the manuscript focuses not only on predictive performance but also on whether the network internalizes physically meaningful depth-related cues.

The main contributions of this work are as follows:

1. A structured tactile dataset for intraoperative-style lung nodule localization, comprising 42 size-depth conditions with three repeated experiments per condition.
2. A mechanism analysis showing that nodule size dominates amplitude-related responses, while depth primarily affects spread, shape, deformation position, and phase-dependent dynamics.
3. A staged learning framework in which raw-input neural networks perform nodule detection, size estimation, and size-aware coarse depth classification.
4. An interpretability pipeline combining XGBoost contribution analysis, latent linear probes, hard-pair analysis, and phase occlusion to identify what the depth classifier has learned.

## 2. Materials and Methods

### 2.1 Experimental setup and data organization
The data were collected from a phantom-based pressing system designed to emulate intraoperative lung palpation. A flexible tactile array recorded frame-wise stress values over a 12 x 8 sensing grid. The complete design matrix consisted of 42 experimental conditions formed by 7 nodule diameters (0.25, 0.5, 0.75, 1.0, 1.25, 1.5, and 1.75 cm) and 6 depths (0.5, 1.0, 1.5, 2.0, 2.5, and 3.0 cm). Each condition was measured three times, corresponding to `1.CSV`, `2.CSV`, and `3.CSV`.

Each CSV stores a time series of 96 tactile values per frame. These values are reshaped into a 12 x 8 stress map. Manual annotation was used to label positive temporal segments, namely frames in which a nodule-related response was considered visible. Group-level labels for size and depth are inherited from the experimental design condition of that recording.

### 2.2 Input unit and task protocol
The formal input unit is a sliding temporal window of length `T = 10` frames with stride `2`, represented as a tensor of shape `10 x 1 x 12 x 8`. Three tasks were defined:

1. **Detection**: determine whether the center frame of the window lies inside a manually labeled positive segment.
2. **Size inversion**: for positive windows, predict both a 7-class size label and a continuous size value in centimeters.
3. **Coarse depth classification**: for positive windows, classify depth into three ordered groups:
   - shallow: 0.5 and 1.0 cm
   - middle: 1.5 and 2.0 cm
   - deep: 2.5 and 3.0 cm

This protocol was kept fixed across the formal experiments.

### 2.3 Split strategy
The paper uses two split protocols because the statistical properties of the tasks differ.

For detection and global development:
- `1.CSV + 2.CSV` were used as the development domain.
- `3.CSV` was held out as the final test repetition.
- Within the development domain, group-level splitting was used to avoid frame leakage.

For the final size-only router and the final depth experiments:
- `1.CSV -> train`
- `2.CSV -> val`
- `3.CSV -> test`

This second protocol is stricter for the depth task because it preserves all depth classes within each evaluation repetition and avoids the degenerate validation behavior observed under condition-held-out splits.

### 2.4 Non-deep-learning mechanism analysis
Before neural modeling, frame-level and window-level statistics were computed to identify candidate size- and depth-sensitive cues. The following groups of descriptors were analyzed:

- amplitude-related features: peak, mean, sum, and high-percentile response
- spread-related features: hotspot radius, area, second-moment spread
- shape and contrast features: center-border contrast, anisotropy, peak count
- distribution-complexity features: spatial entropy
- deformation-position features: centroid row/column and centroid drift
- temporal features: rise time, persistence, decay, and phase-resolved response

The mechanism analysis showed that size is the dominant effect, whereas depth is weaker and condition-dependent. Importantly, depth did not appear as a simple monotonic attenuation of peak amplitude. Instead, its influence was concentrated in spread, shape, deformation position, and phase-related response.

### 2.5 XGBoost baselines
To test whether these physically interpretable features were sufficient for prediction, XGBoost baselines were trained on handcrafted window-level descriptors. Separate models were used for:

- detection
- size classification
- size regression
- coarse depth classification

These baselines serve two purposes. First, they provide a quantitative lower bound for what can be achieved without end-to-end raw-input learning. Second, they expose which feature families dominate the depth decision, giving a reference for later neural interpretability analyses.

### 2.6 Neural network design

#### 2.6.1 Stage 1 detector
The detection model uses a dual-stream spatial-CNN + MS-TCN architecture. One stream consumes the normalized raw window and the other consumes the frame difference sequence. The model outputs a single detection probability.

#### 2.6.2 Dedicated positive-window size router
The final size router is trained only on positive windows. It consumes three complementary signals derived from the same tactile window:

- raw amplitude stream
- normalized shape stream
- temporal difference stream

The output is both a 7-class size prediction and a continuous size estimate. This dedicated router was introduced because a shared multitask size head was found to be the main source of routing error in the depth chain.

#### 2.6.3 Raw-input size-routed depth network
The final depth model also uses raw amplitude, normalized shape, and frame-difference streams, but differs in the output design. Instead of a single global depth head, the model uses **size-routed depth experts**. During training and oracle evaluation, ground-truth size routing is used. During deployment-style evaluation, the route is determined by the dedicated size router.

This architecture was motivated by two observations:

1. coarse depth is learnable from the raw input only when the size-depth coupling is respected;
2. non-size-aware raw end-to-end depth models collapse to majority-class behavior.

### 2.7 Explainability analysis
The neural depth model was interpreted in three complementary ways.

1. **Latent linear probes**: the trained raw-input depth network was frozen and linear ridge probes were trained on its latent representation to recover the same physical descriptors that had been identified in the non-deep-learning analysis.
2. **Hard-pair analysis**: matched pairs of large-deep and small-shallow cases were constructed to test whether the model relied exclusively on peak amplitude.
3. **Phase occlusion**: loading early, loading late, peak neighborhood, and release phases were selectively occluded to measure how much each phase contributed to the probability of the true depth class.

These analyses were designed to answer whether the depth model genuinely learned depth-related cues or simply reused stronger size- or amplitude-related proxies.

### 2.8 Evaluation metrics
Detection was evaluated by AUC, average precision, and F1 at the validation-selected threshold. Size estimation was evaluated by top-1 accuracy, top-2 accuracy, MAE, and median absolute error. Coarse depth classification was evaluated by accuracy, balanced accuracy, and confusion matrices. Because the depth classes are moderately imbalanced, balanced accuracy was treated as the primary depth metric.

## 3. Results

### 3.1 Non-deep-learning analysis established the structure of the task
The mechanism analysis showed that the tactile response contains clear information for nodule detection and size inversion. Positive frames had substantially stronger and more concentrated responses than negative frames. Across the 42 conditions, size exerted a strong and consistent effect on peak amplitude, integrated response, and center-border contrast. Depth also influenced the tactile pattern, but its effect was weaker and not well described by a simple monotonic amplitude drop.

Instead, depth was better characterized by spread, distribution complexity, deformation position, and phase-dependent behavior. This observation defined the core challenge of the paper: depth is a weaker and size-dependent cue, so the neural network must avoid collapsing depth into a size or peak-amplitude surrogate.

### 3.2 XGBoost showed that size and depth information are present in structured physical descriptors
Under the formal protocol, XGBoost reached a test AUC of 0.820, AP of 0.506, and F1 of 0.619 for detection. On ground-truth positive windows, it achieved size top-1 accuracy of 0.670, top-2 accuracy of 0.802, and a size MAE of 0.147 cm. For coarse depth classification, XGBoost reached an accuracy of 0.515 and a balanced accuracy of 0.514, clearly above the majority-class baseline of 0.333.

Feature-contribution analysis showed that the depth baseline relied mainly on deformation-position, shape-contrast, and spread-related feature families, while pure amplitude contributed less. This result established two important facts. First, the dataset contains usable depth information. Second, that information is not dominated by a single peak-amplitude cue.

### 3.3 Raw-input depth classification failed without size-aware decomposition
The first raw-input end-to-end coarse depth model, which did not explicitly separate the size and depth roles, collapsed to majority-class behavior under the correct repetition-wise evaluation protocol. Its test balanced accuracy remained near 0.33. This failure was informative rather than incidental: it indicated that raw input alone is insufficient unless the architecture explicitly accounts for the size-dependent nature of the depth cue.

### 3.4 A size-routed raw-input network recovered coarse depth information
After replacing the single depth head with size-routed depth experts, the raw-input neural network became effective. Under ground-truth size routing, the model achieved a validation balanced accuracy of 0.619 and a test balanced accuracy of 0.512. This result is essentially on par with the XGBoost coarse-depth baseline (0.514), but is obtained directly from raw spatiotemporal tactile windows instead of handcrafted descriptors.

This finding is central to the paper. It shows that coarse depth information is learnable from raw tactile input, but only if the network respects the size-conditioned structure of the task.

### 3.5 The latent representation contained depth-related information beyond size identity
To test whether the raw-input network had internalized physically meaningful depth cues, we froze the trained size-routed model and trained linear probes on its latent representation. Across the selected descriptor set, the mean test `R^2` of the latent probe was 0.366, compared with 0.236 for a size-only probe. The strongest gains over the size-only probe appeared for distribution complexity, temporal-phase descriptors, and deformation-position variables.

Several individual descriptors were recovered particularly well from the latent representation, including spatial-entropy-related measures and selected contrast/spread descriptors. These results indicate that the network was not merely memorizing size identity. Instead, it encoded part of the physically interpretable structure associated with depth variation.

### 3.6 Hard-pair analysis showed partial but incomplete resistance to amplitude-based confounding
We next examined whether the model could distinguish large-deep from small-shallow cases that produced similar peak responses. Eight hard pairs were constructed. In 75% of these pairs, the deeper case received a higher deep-class probability than the matched shallower case. This result suggests that the model was not exclusively relying on amplitude.

However, the matched pairs still had a nontrivial residual difference in peak response, and the evidence should therefore be considered moderate rather than definitive. The current model cannot yet support a strong claim that large-deep versus small-shallow confounding has been fully resolved.

### 3.7 Phase occlusion revealed that the current depth model still over-relies on the peak neighborhood
Phase occlusion further clarified the temporal strategy of the network. Occluding the peak neighborhood caused the largest mean drop in the true-class depth probability (0.162), whereas the drops for loading early and loading late were much smaller (0.057 and 0.041, respectively), and release produced almost no decrease. This is not fully aligned with the non-deep-learning mechanism analysis, which had indicated a more important role for early loading and release dynamics.

Therefore, the neural network has learned part of the relevant depth-related structure, but its temporal use of that structure remains biased toward the peak neighborhood.

### 3.8 The main deployment bottleneck was the upstream size router
Although the raw-input size-routed depth network performed well under oracle size routing, deployment requires predicted size routing. When the older shared multitask size head was used as the router, size top-1 accuracy on Stage 3 positive windows was only 0.523, and predicted-route depth balanced accuracy dropped from 0.524 to 0.441.

A more detailed analysis showed that when the size route was correct, depth balanced accuracy was 0.556, whereas it fell to 0.320 when the route was wrong. This demonstrated that the main bottleneck in the deployed chain was not the depth trunk itself, but the upstream size-routing error.

### 3.9 A dedicated raw-input size router substantially improved the full size-to-depth chain
To address the routing bottleneck, we trained a dedicated size-only router on positive windows using raw amplitude, normalized shape, and frame-difference streams. The model achieved test top-1 accuracy of 0.660, test top-2 accuracy of 0.841, and MAE of 0.291 cm. This performance is close to the XGBoost size baseline and substantially exceeds the shared multitask size head on the same positive-window subset.

Once this dedicated router was used in the depth chain, predicted-route depth balanced accuracy improved to 0.482, reducing the gap to the oracle route. This result clarifies the overall system logic: the depth trunk is viable, but the chain must first stabilize size routing before deployment-quality depth classification can be expected.

## 4. Discussion
The present study shows that tactile localization of lung nodules should not be formulated as a single black-box problem. The three tasks of detection, size inversion, and depth classification occupy different signal regimes. Detection is driven by strong spatiotemporal changes in the tactile map. Size is also a strong effect, reflected in amplitude and central concentration. Depth is weaker, more conditional, and harder to separate from size.

This task structure explains several experimental observations. First, handcrafted descriptors and XGBoost already performed strongly on size and reasonably on depth, demonstrating that the information is present in the data. Second, raw-input neural models only worked for depth after the architecture explicitly imposed size-aware routing. Third, even then, the learned temporal strategy remained biased toward the peak neighborhood, indicating that the network had not yet fully internalized the phase pattern suggested by the physics analysis.

This mismatch between mechanism analysis and neural phase preference is important. It suggests that the present model solves coarse depth mainly by exploiting the most discriminative part of the observed sequence, not necessarily the most physically complete one. From a systems perspective, this is still useful, but from a scientific perspective it sets a clear limit: the current model provides **interpretable coarse classification**, not full physical reconstruction of depth mechanics.

The routing experiments are also instructive. Under oracle size routing, the raw-input depth model is viable. Under predicted routing, performance degrades sharply if size estimation is weak. This means that in a real deployment chain, improving the size router may yield more system-level benefit than further complicating the depth head. The dedicated size router was therefore a necessary step in clarifying the system logic.

Several limitations remain. First, the current depth result is coarse rather than fine-grained. Second, the strongest neural evidence for depth relies on a size-routed architecture and does not yet prove that the network robustly resolves all large-deep versus small-shallow ambiguities. Third, the phase-occlusion analysis indicates that the model still underuses the loading and release phases. Future work should therefore focus on phase-aware supervision, teacher-guided concept regularization, or active-touch protocols that explicitly increase depth sensitivity.

## 5. Conclusion
This work establishes a coherent pipeline for intraoperative-style lung nodule localization from flexible array tactile sensing. Detection and size estimation are strong tasks in this setting, while depth is a weaker but still learnable cue. The key finding is that raw-input neural networks can achieve interpretable coarse depth classification only when the architecture explicitly accounts for the size dependence of depth. Latent probes and controlled explainability analyses further show that the network internalizes part of the physically meaningful depth-related structure, particularly distribution complexity, phase information, and deformation-position cues. The final system logic is therefore clear: detect first, estimate size second, and route depth classification through a size-aware mechanism.
