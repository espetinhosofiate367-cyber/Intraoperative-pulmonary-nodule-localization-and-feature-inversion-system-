# TBME Submission Release v1

This repository is a reviewer-oriented release package for the paper on tactile sensing and deep learning for intraoperative pulmonary nodule localization.

## Start Here

If you only want the fastest route to understand or verify the paper, open:
- `REVIEWER_QUICKSTART.md`

If you want a concise folder map:
- `FOLDER_MAP.md`

If you want the detailed Chinese project navigation:
- `README_CN_PROJECT_GUIDE.md`

## Minimal Review Path

For most reviewers, only these folders matter:
- `01_manuscript/` : current manuscript draft, figure plan, and paper-facing notes
- `03_results_core/` : final result summaries used in the paper
- `04_core_models/` : released checkpoints for the final pipeline
- `05_core_code/` : minimal code needed to inspect or reproduce the main pipeline

Everything else is supportive material or archived context.

## What This Package Optimizes For

This release is organized to make three tasks easy:
1. Read the paper story quickly
2. Verify the reported metrics from saved summaries and checkpoints
3. Re-run the core pipeline with the released code

It is not meant to expose the entire historical development archive.
