# Supporting Notes

这个目录保存的是**仍然有用、但不需要每天打开**的论文支撑资料。

## 适合放在这里的文件类型
- 方法细节蓝图
- 标签协议与任务定义
- 结构化 baseline / explainability 写作笔记
- TBME 投稿补强材料
- 风格参考与引用核验清单
- 模型优化过程记录
