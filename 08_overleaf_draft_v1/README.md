# Overleaf Draft V1

This folder is an Overleaf-ready English draft skeleton for the TBME-target manuscript.

## Goal
- Provide a clean English paper structure based on `MANUSCRIPT_DRAFT_V10_CN.md`
- Show how figures, tables, and references can be organized in Overleaf
- Make it easier to start formal TBME writing without rebuilding the project layout from scratch

## Included
- `main.tex`: paper entry
- `references.bib`: initial bibliography
- `sections/`: section-wise manuscript draft
- `tables/`: ready-to-input LaTeX tables
- `figures/`: selected PDF figures copied from the figure pack

## Notes
- This is a writing skeleton, not yet a polished final English submission.
- Some figure assignments are provisional and can be updated after final figure selection.
- Citation keys are already wired into the text and can be refined later.

## Suggested Overleaf workflow
1. Upload this whole folder as a new Overleaf project.
2. Set the main file to `main.tex`.
3. Use `IEEEtran` with BibTeX.
4. Replace placeholder author names and affiliations.
5. Continue polishing section language directly in `sections/`.
