# 基于柔性阵列触觉传感与层级时空神经网络的术中肺结节探测与特征反演系统

## 摘要
微创胸腔镜和机器人辅助手术显著降低了肺结节切除的创伤，但也使术者失去了开放手术中的直接触觉反馈，从而增加了小结节、深部结节和隐匿性结节的术中探测与定位难度。围绕这一临床问题，本文以**可靠术中探测**为首要目标，并进一步探索结节大小反演和深度粗粒度判别的可行性，构建了一套基于柔性阵列触觉传感与层级时空神经网络的术中肺结节探测与特征反演系统。研究首先在离体猪肺模型上采集 7 个结节大小、6 个埋藏深度和 3 次重复构成的时序触觉数据，并通过机制分析提取扩散、形态、分布复杂度和时序阶段等结构化特征。随后，利用 XGBoost 建立结构化基线，验证触觉数据中不仅包含稳定的结节探测信息，还包含可学习的大小信息和一定程度的粗深度信息。其后，本文构建面向原始时空输入的层级神经网络系统：阶段一完成结节检测，阶段二完成大小分类与回归，阶段三在大小条件下执行粗深度分类，并进一步构建统一层级反演器以直接优化真实 predicted-route 条件下的深度性能。实验结果表明，阶段一检测模型在独立测试集上的 AUC 为 0.8383，高于 XGBoost 的 0.8199；XGBoost 大小反演性能达到 top-1 0.6701、MAE 0.1472 cm，并在深度粗分类上取得 0.5138 的平衡准确率；若不显式处理 size-depth 耦合，原始输入深度网络会退化到多数类附近，而在 size-aware 组织下，原始输入神经网络和统一层级反演器分别将 ground-truth route 与 predicted-route 深度平衡准确率提升至 0.5238 和 0.5337。进一步的 latent probe、branch ablation、hard-pair analysis 与 phase occlusion 结果表明，神经网络内部确实编码并利用了部分与大小和深度相关的物理结构，而非仅依赖单点峰值强度。研究结果说明，柔性触觉时空数据首先可支持可靠术中结节探测，其次能够支持大小反演，而深度信息则可在显式层级组织和可解释性约束下实现粗粒度辨别，为构建可部署、可解释的术中触觉导航系统提供了技术基础。

**关键词**：肺结节探测；柔性触觉阵列；术中导航；大小反演；粗深度分类；可解释人工智能

## 1 引言
随着低剂量计算机断层扫描在肺癌早期筛查中的广泛应用，临床上对微小肺结节和磨玻璃结节的检出率持续升高。对于此类疑似早期病灶，电视辅助胸腔镜手术和机器人辅助手术因其创伤小、恢复快和住院周期短，已成为重要的外科干预方式。然而，微创手术在带来创伤获益的同时，也引入了一个尚未被彻底解决的临床难题：术者在依赖细长器械操作时，几乎完全失去了开放手术中可直接用于识别病灶的触觉反馈。对于深部、小尺寸或隐匿性结节，这种触觉缺失会显著增加漏检、定位不准和扩大切除范围的风险。

目前临床上常见的辅助定位方案，包括术前 CT 引导钩线或染料定位、术中超声以及有限度的数字触诊，但这些方法均存在明确短板。术前侵入式定位具有气胸、出血和标记移位等风险；术中超声对探头贴合、空气干扰控制和操作者经验依赖较强；数字触诊虽然直观，但高度主观，缺乏可量化输出。因此，术中定位真正需要的并不是另一个“辅助视觉手段”，而是一种能够在微创条件下恢复缺失触觉、并将其转化为客观定量信息的技术体系。

柔性阵列触觉传感器为这一目标提供了有前景的路径。与传统单点力传感器不同，柔性阵列可以在按压过程中连续记录肺表面的二维应力分布，并形成具有时间演化特征的时空触觉张量。当高刚度结节埋藏在软组织内部时，结节与周围组织的力学不匹配会改变表面应力场的峰值、扩散范围、空间形态以及时间响应模式。理论上，这些表面信号中同时包含三个层次的信息：首先是**结节是否存在**，其次是**结节大小**，最后才是更弱、更难的**埋藏深度**。

然而，将表面触觉分布直接反演为内部结节参数，本质上是一个病态逆问题。一个浅层但较小的结节，可能在某一按压时刻产生与一个深层但较大的结节相似的表面热点；如果只依赖单帧应力图或简单幅值阈值，模型极易将大小效应误写成深度效应，反之亦然。因此，本文不将“检测、大小、深度”视作同等难度的平行输出，而是将其组织为一个由易到难、由强信号到弱信号的层级问题：**先回答有没有结节，再回答结节有多大，最后在大小条件下探索深度是否能够被粗粒度辨别。**

基于这一认识，本文采用“**机制分析 -> 结构化基线 -> 原始输入神经网络 -> 可解释性验证**”的研究路径。首先，通过统计与物理启发特征分析，回答数据中是否存在与探测、大小和深度相关的结构化信号；其次，利用 XGBoost 建立结构化基线，验证这些信号是否足以支撑检测、大小反演和粗深度分类；随后，在结构化基线的基础上引入神经网络，不再直接喂入手工设计特征，而是让模型从原始时空触觉窗口中自动学习这些信息；最后，通过可解释性分析检验神经网络是否确实捕捉到了与大小和深度相关的部分物理特征，而不是仅仅依赖黑箱相关性完成预测。

需要强调的是，本文引入 XGBoost 并不是为了与神经网络形成简单的替代关系。XGBoost 在本研究中承担的是三个角色。第一，它是**结构化机制基线**，用于回答“数据里是否真实存在可学习的信息”；第二，它是**解释锚点**，帮助我们识别与深度相关的扩散、形态、分布复杂度和时序阶段特征家族；第三，它是**系统比较参照**，使我们能够区分“神经网络是否真正从原始输入中学到了更多”与“神经网络是否只是换了一种更复杂的拟合器”。神经网络的引入，则主要服务于结构化基线无法自然完成的任务：从原始时空输入中自动学习时序模式、在真实 `size -> depth` 路由误差存在时保持系统鲁棒性，并最终支撑术中实时部署。

因此，本文希望回答的不是“神经网络是否全面替代结构化方法”，而是三个更具体的问题：  
1. 柔性触觉时空数据能否首先支持稳定的术中结节探测？  
2. 在探测基础上，是否可以可靠反演结节大小，并进一步粗略区分深度？  
3. 当神经网络在原始时空数据上优于结构化基线后，它内部是否确实编码了部分与大小和深度相关的物理结构？

围绕上述问题，本文的主要贡献概括如下：

1. 构建了面向术中肺结节触觉探测与反演的结构化时空数据体系。在 7 个结节大小、6 个埋藏深度和 3 次重复实验条件下，获得了 126 组逐帧应力分布序列，并据此建立了与检测、大小反演和粗深度分类相对应的窗口级标签协议。

2. 通过机制分析和 XGBoost 基线，证明了柔性触觉时空数据中既存在稳定的结节探测信息，也包含强大小信息和可学习的粗深度信息，从而为后续神经网络建模提供结构化参照。

3. 提出了以检测为核心的层级神经网络体系，将任务组织为“Detection -> Size -> Depth”，并通过统一层级反演器直接优化真实 predicted-route 条件下的系统级深度性能。

4. 通过 latent probes、branch ablation、hard-pair analysis 与 phase occlusion，建立了从结构化特征、XGBoost 解释到神经网络内部表征的可解释性闭环，验证神经网络确实捕捉到了部分与大小和深度相关的物理特征。

## 2 材料与方法

### 2.1 实验平台与数据协议
本研究使用柔性阵列触觉传感系统对离体猪肺模型进行动态按压实验。每一帧包含 96 个传感通道，经重排后形成 12×8 的二维应力矩阵。实验设计矩阵由 7 个结节大小（0.25、0.5、0.75、1.0、1.25、1.5、1.75 cm）和 6 个埋藏深度（0.5、1.0、1.5、2.0、2.5、3.0 cm）构成，共 42 个物理条件；每个条件进行 3 次独立重复，最终得到 126 组带有完整时间演化信息的逐帧触觉序列。

为适配模型输入，本文采用长度为 10 帧、步长为 2 的滑动窗口作为基本样本单位。检测任务以窗口中心帧是否位于人工标注的阳性片段内作为标签；大小和深度标签则继承自对应实验记录的物理条件。检测任务使用 `1.CSV + 2.CSV` 作为开发集、`3.CSV` 作为独立测试集；大小与深度反演任务采用更严格的重复划分协议，即 `1.CSV` 训练、`2.CSV` 验证、`3.CSV` 测试，并仅在真实阳性窗口上训练。

### 2.2 机制分析与结构化物理特征
在进入深度学习之前，本文首先对帧级和窗口级触觉特征进行系统分析。特征包括峰值强度、积分响应、中心-边缘对比、热点面积、热点半径、二阶矩扩散、空间熵、质心位置及漂移、峰值到达时间、衰减特征以及多种窗口级时序统计量。分析结果表明，结节大小是最强主效应，主要影响峰值、整体能量和中心集中度；深度则是更弱但条件依赖的效应，主要体现为扩散范围、分布复杂度、形变位置和按压阶段响应。

这一分析有两个目的。第一，验证数据中是否真实存在可区分的结构化信息；第二，为后续的结构化基线和神经网络解释提供候选物理特征家族。本文将这些特征大致归纳为 amplitude、spread extent、shape contrast、deformation position 和 temporal phase 五类。

### 2.3 XGBoost 结构化基线
在机制分析的基础上，本文采用 XGBoost 分别构建检测、大小分类/回归和深度粗分类基线。XGBoost 基线并不承担最终部署角色，而是用于回答两个更基础的问题：  
1. 这些结构化特征是否足以支撑检测、大小反演和深度粗分类？  
2. 哪些特征家族在深度判别中真正起主导作用？

与直接进入神经网络相比，结构化基线能更早回答“信息是否存在且可学”，从而避免把全部结论建立在黑箱模型之上。与此同时，XGBoost 的特征贡献结果也为后续神经网络设计提供了结构化约束，尤其是在深度主要依赖 spread、shape contrast、deformation position 而非单一峰值幅值这一点上。

### 2.4 层级时空神经网络系统

#### 2.4.1 阶段一：结节探测
检测是本文的首要任务。阶段一检测器直接从原始时空触觉窗口中判断“当前窗口下方是否存在结节”。为此，本文采用双流时空网络，对原始窗口和相邻帧差分窗口并行编码；空间维通过轻量卷积提取，时间维通过 MS-TCN 聚合，最终输出结节存在概率。该阶段的目的不是反演全部物理参数，而是先建立一个稳定可靠的术中探测器，为后续反演提供高置信入口。

#### 2.4.2 阶段二：大小反演
在真实阳性窗口上，系统进一步执行大小反演，输出 7 类大小分类和连续大小回归。由于前期分析表明大小是强主效应，且也是深度任务的路由条件，本文为大小任务单独训练专用 router，而不是将其简单附属于共享多任务头。这样做的目标是提升 route quality，并减少上游错误向深度任务传播。

#### 2.4.3 阶段三：大小条件下的粗深度分类
深度任务被定义为 `shallow / middle / deep` 三类粗分类，而非连续深度回归。更重要的是，本文并未采用一个全局共享的深度头，而是使用 size-routed depth experts。其核心假设是：深度效应并非独立于大小存在，而是在不同大小条件下表现出不同的扩散和时序模式。因此，深度必须在大小条件下建模，而不是与大小平级并列输出。

#### 2.4.4 统一层级反演器
为了进一步优化真实部署链路，本文构建了统一层级反演器。该模型联合输入原始幅值序列、归一化形态序列、差分序列以及筛选后的结构化物理特征，并同时输出 size classification、size ordinal prediction、size regression 和 routed depth。与早期“先训 size，再外接 depth expert”的松耦合串联方式不同，统一层级反演器在训练中显式加入 ground-truth route、hard predicted route、soft predicted route、top2 route 和 route consistency 目标，从而直接优化真实 predicted-route 条件下的系统级深度性能。

### 2.5 可解释性分析与系统落地
为了回答神经网络是否真正学到了与大小和深度相关的物理结构，本文从三个层面开展解释性分析：  
1. 机制先验层：统计分析确定候选深度敏感特征；  
2. 结构化基线层：XGBoost 特征贡献揭示哪些特征家族主导粗深度分类；  
3. 神经网络内部表征层：通过 latent linear probes、branch ablation、hard-pair analysis 和 phase occlusion 评估网络是否编码并利用了超出大小身份本身的深度相关结构。

在工程实现上，检测结果始终显示，大小和深度结果仅在检测概率超过阈值后输出，并最终集成到术中实时可视化界面中。

## 3 结果

### 3.1 机制分析与 XGBoost 基线先证实了“可探测、可反演、可粗分深度”的信息层级
结构化特征分析首先表明，结节是否存在、结节大小和埋藏深度在触觉数据中的表现强度并不相同。其中，检测相关差异最显著，大小是强主效应，而深度是更弱、更容易被大小掩盖的条件依赖信号。XGBoost 基线进一步将“信息存在”转化为“信息可学”：在测试集上，XGBoost 的检测 AUC 为 0.8199，大小反演达到 top-1 0.6701、top-2 0.8023 和 MAE 0.1472 cm，而深度粗分类平衡准确率达到 0.5138，显著高于多数类基线的 0.3333。这个结果非常关键，因为它说明深度并非完全不可学，只是相比检测和大小更弱、更依赖正确的任务组织。

### 3.2 检测是最稳健、也是最应优先优化的主任务
在独立测试集 `3.CSV` 上，阶段一 raw+delta 检测器取得 AUC 0.8383、AP 0.5357 和 F1 0.6216，整体优于 XGBoost 的 0.8199、0.5063 和 0.6185。该结果表明，在“当前窗口下方是否存在结节”这一主问题上，原始时空神经网络已经展现出优于结构化树模型的建模能力。对本文而言，这一结果的意义不仅在于分数提升，更在于它支撑了整个系统的设计前提：术中系统首先需要的是一个稳定的探测器，而不是一开始就追求复杂的深度反演。

### 3.3 大小反演是检测之后最强、最稳定的 secondary outcome
在真实阳性窗口上，XGBoost 保持了目前最优的 standalone 大小回归表现，MAE 为 0.1472 cm。专用 raw-input size-only router 在分类上已逼近并部分超过结构化基线，其 top-1 和 top-2 分别达到 0.6600 和 0.8405，但 MAE 为 0.2907 cm；统一层级反演器在整体链路优化下的大小表现为 top-1 0.6180、top-2 0.7859 和 MAE 0.1565 cm。这个结果说明，大小确实是强信号，但不同架构在“独立 size 最优”和“整条链路最优”之间存在明确权衡：如果单独追求 size 数值最优，结构化基线仍具有优势；但如果目标是为后续 depth 路由服务，则统一层级模型提供了更好的系统级平衡。

### 3.4 深度并非不可学，但如果不显式处理 size-depth 耦合，原始输入网络会退化到多数类附近
最初的 raw-input end-to-end 深度模型在严格重复划分协议下表现接近多数类基线，测试平衡准确率约为 0.33。这一结果并不意味着原始时空触觉数据不能支持深度学习，而是更清楚地揭示了本文的核心困难：当大小是强主效应、深度是弱主效应时，如果仍采用普通共享 depth head，网络会优先吸收强度和大小相关信号，从而使深度判别被更强因素淹没。

当单一深度头被替换为 size-routed depth experts 后，情况发生显著变化。在 ground-truth route 条件下，raw-input size-routed 深度网络的测试平衡准确率达到 0.5238，已经与 XGBoost 的 0.5138 相当并略有超过。该结果证明，原始输入神经网络并非学不会深度，而是必须在大小条件下组织深度任务，才能恢复可用的 coarse depth information。

### 3.5 真正阻碍部署的不是“深度信息不存在”，而是 `size -> depth` 路由误差
尽管 size-routed 深度网络在 oracle route 条件下表现良好，但真实部署必须使用预测 size route。实验表明，若采用早期共享多任务 size head 进行路由，则 Stage3 正窗上的 size top-1 仅约为 0.5226，predicted-route 深度平衡准确率下降至 0.4822。更进一步的拆分分析显示：当 route 正确时，depth 的平衡准确率可达 0.5557；当 route 错误时，该值降至 0.3204。这个现象说明，当前链路的主要误差传播源并不是 depth trunk 本身，而是上游 size router 的错误将样本送入了错误的深度专家。

### 3.6 统一层级反演器在真实 predicted-route 条件下首次稳定超过结构化基线
针对上述瓶颈，本文进一步采用统一层级反演器，将大小判别、路由策略和深度专家放入同一训练图中协同优化。结果显示，该模型在测试集上的 ground-truth route 深度平衡准确率达到 0.5407，而在更关键的 predicted-route hard balanced accuracy 上达到 0.5337，不仅明显高于旧 predicted-route 链路的 0.4822，也稳定超过了 XGBoost 的 0.5138。这个结果使本文的系统级故事真正闭合：结构化基线先证明深度信息存在，原始输入神经网络进一步在层级组织和 route-aware 优化下超越 baseline，且这一优势出现在**真实部署条件**而不仅是理想 oracle 条件下。

### 3.7 可解释性结果表明，神经网络确实捕捉到了部分与大小和深度相关的物理特征
为验证最终模型不是仅凭黑箱相关性做出判断，本文进一步对原始输入 size-routed 网络和统一层级反演器开展解释性分析。对于 raw-input size-routed 深度网络，latent probe 的平均测试 `R^2` 为 0.3657，而仅以 size identity 为输入的 probe 为 0.2356，说明网络内部已编码了一部分超出大小本身的深度相关结构。对于统一层级反演器，`hybrid latent` 的平均测试 `R^2` 进一步提升至 0.5512，明显高于 `raw trunk` 的 0.4226 和 `size-only probe` 的 0.2985，表明最终模型的内部表示同时整合了原始时空线索和结构化物理线索。

branch ablation 结果进一步说明了模型依赖的信息来源。相对于 full model 的 size top-1 0.6180 和 depth hard-route bAcc 0.5337，去除 `raw`、`shape`、`delta` 和 `tabular` 分支都会导致性能下降，其中 `shape` 与 `tabular` 分支对最终 depth 判别影响最大。hard-pair analysis 显示，在构造的困难样本对中，更深样本在 80% 的配对中获得更高的 `p_deep`，说明模型并非只依赖单点峰值幅值完成深度判断。phase occlusion 则显示，当前模型在时间维上的主要依赖仍集中在 `peak neighborhood`，而 `loading early` 与 `release` 的贡献相对较弱。这意味着神经网络已经捕捉到部分与大小和深度相关的物理特征，但尚未完全恢复机制分析所提示的全部时序结构。

### 3.8 实时界面验证了该体系具备术中系统化使用潜力
在工程层面，本文已将阶段一检测器与统一层级反演器集成到实时可视化界面中。界面可以同步显示原始触觉图、检测概率、大小分布和深度分布，并遵循“检测始终显示、大小与深度仅在高置信条件下输出”的逻辑。这一点说明本文并非只停留在离线模型比较，而是已经将“结构化机制发现 -> 原始输入学习 -> 真实链路优化 -> 术中显示”串接为一个可部署的系统原型。

## 4 讨论
本文最重要的结论不是“神经网络在所有指标上都压过了 XGBoost”，而是更具体也更符合临床需求的三点。第一，柔性触觉时空数据首先能够支持**稳定的结节探测**，这是最核心、也最有直接临床意义的结果。第二，在探测基础上，数据中确实还包含较强的大小信息以及较弱但可学习的深度信息。第三，深度信息的利用并不能通过一个平级多头网络自然获得，而必须通过显式的层级任务组织、大小感知路由以及对真实误差传播的直接优化来实现。

从这个角度看，XGBoost 与神经网络并不是竞争关系，而是递进关系。XGBoost 首先回答“数据里有没有可学习的信息、哪些物理特征在起作用”；神经网络随后回答“在不直接输入这些手工特征的情况下，原始时空触觉窗口能否自动恢复这些信息，并在真实部署中形成更好的系统表现”。因此，本文并不是“先试一个传统模型，再换一个更复杂模型”，而是依次建立：机制认识、结构化基线、原始输入学习和模型解释四个层次的证据链。

本文还揭示了一个对术中系统设计非常重要的事实：单点的最佳中间任务指标，并不等价于系统级最优。以大小任务为例，XGBoost 仍保持最优 MAE，但统一层级反演器在真实 predicted-route 深度上获得了更优结果。这表明，对于最终系统而言，更值得优化的不是某一个孤立任务的数值上限，而是整条推理链在真实误差传播条件下的鲁棒性。换言之，本文真正优化的对象不是“最好的 size 模型”或“最好的 depth 头”，而是“最可靠的 detection-first 术中决策链路”。

关于深度问题，本文当前的证据支持将其定位为**coarse depth discrimination**，而不是连续深度反演。一方面，XGBoost 和 size-routed 神经网络都已证明深度信息客观存在且可以被粗粒度辨别；另一方面，phase occlusion 和 hard-pair 分析也提示，当前模型虽然不再只是依赖峰值幅值，但在时间维上仍偏向 `peak neighborhood`，对 `loading/release` 阶段的利用仍不足。因此，深度相关图和结果在主文中应主要承担“给出证据”的职责，而它为何仍然困难、为何尚未完全恢复全部物理机制，则更适合在 Discussion 中解释和界定边界。

本研究仍存在若干局限。第一，统一层级反演器虽然在 predicted-route 深度上超过了 XGBoost，但其 standalone size 指标尚未在所有维度上全面超过结构化基线。第二，本文的深度任务目前仍是三类 coarse depth classification，而非更精细的连续深度定量。第三，尽管解释性分析已支持神经网络确实编码了部分与大小和深度相关的结构，但这些结果仍应被理解为“部分机制一致性证据”，而非对完整生物力学规律的最终恢复。未来工作需要继续强化 phase-aware 建模、设计更针对 routing noise 的训练策略，并在更接近真实临床场景的数据上进一步验证系统的泛化能力。

## 5 结论
本文围绕术中肺结节触觉缺失问题，提出了一条以**可靠探测为核心**、逐步扩展到大小反演和粗深度辨别的研究路径。结果表明：柔性触觉时空数据首先能够支持稳定的结节探测；在此基础上，结节大小可被较可靠地反演，而深度信息则作为更弱、更依赖层级组织的条件信号被粗粒度辨别。通过机制分析和 XGBoost 基线，本文先证明了这些信息的存在与可学习性；通过层级神经网络和统一层级反演器，进一步证明原始时空输入可以在真实 predicted-route 条件下实现超过结构化基线的系统级性能；通过可解释性分析，则验证神经网络内部确实编码并利用了部分与大小和深度相关的物理结构。整体而言，本文构建的是一个“检测优先、反演增强、可解释验证、面向部署”的术中触觉导航框架，而不是单纯追求深度反演分数的模型比较。这一框架为柔性触觉传感在术中肺结节探测与导航中的应用提供了一个更具工程可行性和临床可沟通性的基础。
