# 基于柔性阵列触觉传感与层级时空神经网络的术中肺结节定位与特征反演系统

## 摘要
随着低剂量计算机断层扫描在肺癌早筛中的普及，临床上对微小肺结节和磨玻璃结节的检出率持续升高。对于此类高度疑似早期病灶，电视辅助胸腔镜手术和机器人辅助手术虽然显著降低了创伤，但也使术者失去了开放手术中至关重要的直接触觉反馈，进而增加了深部、小尺寸和隐匿性结节在术中定位中的困难。为恢复和量化这一触觉信息，本文构建了一套基于柔性阵列应力传感与层级时空神经网络的术中肺结节定位与特征反演系统。系统以时序应力图窗口作为输入，依次执行结节检测、大小反演和大小感知深度粗分类，并将结果实时加载到可视化界面中。针对深度信息弱、与大小高度耦合且易受路由误差影响的问题，本文先通过非深度学习机制分析和 XGBoost 基线识别出深度相关的扩散、形态、分布复杂度和时序阶段特征，再据此设计 size-routed 深度网络及统一层级反演器。实验结果表明，阶段一检测模型在独立测试集上的 AUC 为 0.8383，优于 XGBoost 的 0.8199；统一层级反演器在预测路由条件下的深度粗分类平衡准确率达到 0.5337，高于 XGBoost 的 0.5138；原始输入神经网络在 size-aware 条件下能够学习粗深度信息，并通过 latent probe、hard-pair 和 phase occlusion 形成可解释性闭环。研究表明，原始时空触觉数据中确实包含可学习的深度信息，但其稳定利用依赖于层级任务组织、大小感知路由以及对真实部署误差传播的显式优化。

**关键词**：肺结节定位；柔性阵列传感器；触觉反演；层级时空神经网络；可解释人工智能；术中导航

## 1 引言
随着低剂量计算机断层扫描在肺癌早期筛查中的广泛应用，临床上对无症状微小肺结节及磨玻璃结节的检出率呈持续上升趋势。对于此类高疑似早期恶性病灶，电视辅助胸腔镜手术和机器人辅助手术因其创伤小、恢复快等优势，已逐渐成为临床主流干预方式。然而，微创技术在降低创伤的同时，也使外科医生丧失了开放手术中极其重要的直接触觉反馈能力。对深部结节、小结节以及缺乏表面牵拉征象的病灶而言，这种触觉缺失会显著增加术中漏检、扩大切除范围和中转开胸的风险。

在现有临床实践中，术者常通过数字触诊、术前 CT 引导定位、染料标记、金属钩定位或术中超声等方式辅助定位病灶，但这些方案各有局限。数字触诊强烈依赖操作者经验，缺乏客观量化标准；术前侵入式定位存在气胸、出血、标记移位等并发症风险；术中超声则对探头贴合、成像经验和操作路径提出较高要求。归根结底，当前术中定位技术尚未形成一种既能保留微创优势、又能稳定恢复“触觉信息”的实时定量方案。

柔性触觉传感器阵列为解决这一问题提供了一条有前景的技术路线。相较于传统单点力传感器，柔性阵列能够在按压过程中连续采集表面二维应力分布，并进一步形成具有时间演化信息的时空应力张量。对于肺结节这类深埋于软组织内部的高刚度包体，当柔性传感器对肺组织施加接触压力时，结节及其周围组织的机械不匹配将改变表面应力的峰值、扩散、形态和演化速度。理论上，这种表面响应包含病灶是否存在、病灶大小以及病灶埋藏深度的关键信息。

然而，将表面触觉矩阵进一步反演为内部结节参数，本质上是一个病态逆问题。一个浅层但较小的结节，可能在某些按压阶段产生与一个深层但较大的结节相似的峰值响应；若仅依赖单帧应力图或简单强度阈值，模型容易把深度效应误判为大小效应，或者反过来把大小差异误认为深度差异。更进一步，在医疗场景中，仅仅得到“可用的分类结果”并不足够，模型还必须具备一定的物理可解释性，才能回答“为什么网络做出了这样的深度判断”，并为临床使用建立可信基础。

围绕这一问题，本文采用了“先机制分析，再层级建模，再解释验证”的研究路径。首先，我们在 42 个大小-深度组合、每组 3 次重复的动态按压实验中，系统分析了结节存在、大小和深度对表面应力响应的影响。结果显示，结节大小是强主效应，主要影响峰值强度、整体能量和中心集中度；深度则是更弱、更条件依赖的效应，主要体现为扩散范围、分布复杂度、形变位置以及按压阶段相关的时序变化。随后，我们建立 XGBoost 基线，验证这些结构化物理特征中已包含可学习的大小和深度信息。最后，在此基础上设计原始输入神经网络，并逐步证明：若不显式考虑 size-depth 耦合，网络会在深度任务上退化到多数类附近；只有将深度建模为大小条件下的细粒度判别问题，原始输入神经网络才能稳定学习粗深度信息。

从写作思路上，本文并不把“检测、大小、深度”视作三个彼此独立的平级任务，而是将其组织为一个逐级细化的层级推理系统。该思想与层级学习框架中的“先粗后细”相一致，但具体实现上并未照搬病理全切片图像中的 bag-instance 多实例学习结构，而是结合触觉时序任务特点，将层级关系落实为：首先完成结节检测，其次进行大小反演，最后在大小条件下执行深度粗分类，并通过 route consistency 约束真实部署中的误差传播。这样做的核心目的，是避免网络把“大而深”简单识别为“小而浅”，并使深度判别真正建立在扩散、形态和时序机制之上。

本文的主要贡献可以概括为以下四点：

1. 构建了面向术中触觉定位的结构化时空数据体系。在 7 个结节尺寸、6 个埋藏深度和 3 次重复实验的设计矩阵下，获得了 126 组具有真实时序特征的逐帧应力分布序列，并完成了与窗口级检测、大小和深度任务相匹配的标签协议设计。

2. 提出了一套层级式术中肺结节反演系统。系统由阶段一检测器、阶段二大小反演器和阶段三大小感知深度分类器组成，并进一步构建统一层级反演器，直接优化预测路由条件下的深度性能。

3. 证明了原始输入神经网络可以学习粗深度信息，但前提是显式尊重深度的大小依赖结构。在 ground-truth size route 条件下，原始输入的 size-routed 深度网络已可达到与 XGBoost 基线相当的性能；在进一步考虑真实预测路由后，统一层级反演器将 predicted-route 深度性能稳定推过了 XGBoost。

4. 建立了从机制分析、基线解释到神经网络解释的可解释性闭环。通过 XGBoost 特征贡献分析、latent linear probes、hard-pair analysis 与 phase occlusion，本文不仅回答了“模型能否判别深度”，也回答了“模型主要依赖哪些深度相关特征以及仍存在哪些偏差”。

## 2 材料与方法
### 2.1 数据采集与实验设计
本研究使用柔性阵列触觉传感系统对离体猪肺模型进行动态按压实验。每一帧包含 96 个传感通道，经重排后形成 12 × 8 的应力分布图。实验设计矩阵由 7 个结节尺寸（0.25、0.5、0.75、1.0、1.25、1.5、1.75 cm）和 6 个埋藏深度（0.5、1.0、1.5、2.0、2.5、3.0 cm）组成，共 42 个条件，每个条件进行 3 次独立重复测试。最终形成 126 组带有完整时序特征的逐帧应力数据序列。

为适配神经网络输入，本文采用长度为 10 帧的滑动窗口作为基本样本单位，步长为 2。检测任务采用窗口中心帧是否位于人工标注阳性片段内作为标签；大小与深度标签由该记录的实验条件继承而来。检测任务采用 `1.CSV + 2.CSV` 开发、`3.CSV` 测试的协议，而大小与深度反演任务采用更严格的重复划分，即 `1.CSV` 训练、`2.CSV` 验证、`3.CSV` 测试，并仅在阳性窗口上训练。

### 2.2 非深度学习机制分析与结构化基线
在神经网络训练之前，本文首先对帧级和窗口级应力特征进行了系统分析。统计特征包括峰值强度、积分响应、中心-边缘对比、热点面积、热点半径、二阶矩扩散、空间熵、质心漂移、峰值到达时间、衰减特征以及若干窗口级时序统计量。分析结果表明，大小是强主效应，而深度主要影响扩散、分布复杂度、形变位置和时序阶段响应。

在此基础上，本文建立了 XGBoost 基线模型，用于完成检测、大小分类/回归和深度粗分类任务。该基线的作用有两点：其一，验证数据中确实存在可学习的大小与深度信息；其二，为后续神经网络的可解释性提供结构化参照。实验结果显示，XGBoost 在检测任务上略弱于最优神经网络，但在深度粗分类任务上可达到 0.5138 的平衡准确率，说明深度信息虽然较弱，但并非不可学习。

### 2.3 层级时空神经网络系统
本文将整套系统组织为三级层级推理过程。

#### 2.3.1 阶段一：结节存在检测
检测阶段使用双流时空网络，对原始窗口和相邻帧差分窗口进行并行编码。空间维由轻量卷积编码器提取，时间维通过 MS-TCN 聚合，最终输出结节存在概率。该设计的目标是先稳定回答“当前窗口下方是否存在结节”，以便后续反演仅在高置信窗口上执行。

#### 2.3.2 阶段二：大小反演
大小反演阶段仅在阳性窗口上运行。网络直接从原始时序应力窗口学习 7 类大小分类和连续大小回归，并作为后续深度判别的路由条件。由于前期分析已表明大小是强主效应，因此本文为大小任务单独训练专用 router，以减少共享多任务头带来的误差传播。

#### 2.3.3 阶段三：大小感知深度粗分类
针对深度任务，本文并未采用单一共享的深度分类头，而是使用 size-routed depth experts。该设计的出发点在于：深度效应并不是全局一致的独立信号，而是在不同大小条件下呈现出不同的应力扩散和时序模式。因此，网络首先依据大小信息选择对应的深度专家，再进行 `shallow / middle / deep` 的粗分类判别。

#### 2.3.4 统一层级反演器
为了进一步提高真实部署条件下的性能，本文构建了统一层级反演器。该模型联合输入原始幅值序列、归一化形态序列以及筛选后的结构化物理特征，并同时输出大小分类、大小有序预测、大小回归和 routed depth。与早期的“先 size 再 depth”的松耦合串联方式不同，统一层级反演器在训练中显式加入了 ground-truth route、hard predicted route、soft predicted route、top2 route 以及 route consistency 目标，从而直接优化真实部署链路中的 predicted-route depth 表现。

### 2.4 可解释性分析框架
为了回答“网络究竟学到了哪些深度相关信息”，本文从三个层面构建解释性框架。

1. 机制先验层：利用非深度学习统计分析明确候选深度敏感特征，包括扩散、形态、位置和阶段响应。
2. 结构化基线层：通过 XGBoost 特征贡献分析验证哪些特征家族主导了深度判别。
3. 神经网络内部表征层：对训练后的深度网络执行 latent linear probes、hard-pair analysis 和 phase occlusion，以判断其内部表征是否真正编码了超出大小本身的深度相关结构。

## 3 结果
### 3.1 机制分析显示深度主要影响扩散、形态与阶段响应，而非简单峰值衰减
在所有条件组的帧级和窗口级统计分析中，结节大小与峰值强度、积分响应和中心集中度之间表现出较强而稳定的关联。相比之下，深度与峰值幅值之间的全局相关性较弱，并不支持“越深越弱”的单调线性解释。更合理的结论是，深度主要通过改变热点扩散范围、分布复杂度、形变位置以及按压过程中的阶段性时序响应来影响表面应力分布。这一结果定义了本文后续网络设计的核心约束：深度是一个弱、条件依赖、大小耦合的信号。

### 3.2 XGBoost 证明结构化物理特征中已包含可学习的大小和深度信息
在正式协议下，XGBoost 检测基线在测试集上取得 AUC 0.8199、AP 0.5063 和 F1 0.6185。对真实阳性窗口，其大小 top-1 准确率为 0.6701，top-2 为 0.8023，大小 MAE 为 0.1472 cm；深度粗分类准确率为 0.5151，平衡准确率为 0.5138。特征贡献分析进一步表明，深度任务主要依赖 deformation-position、shape-contrast 和 spread-related feature families，而纯 amplitude family 的贡献较低。该结果明确说明：数据中并不缺少深度信息，真正的挑战在于神经网络是否能够从原始输入中提取和组织这些信息。

### 3.3 若不显式处理 size-depth 耦合，原始输入深度网络会退化到多数类附近
最初的原始输入深度分类网络在正确的重复划分评估协议下表现接近多数类基线，其测试平衡准确率约为 0.33，几乎等同于随机偏置分类。这一结果具有重要方法学意义：它表明失败并非来自“原始输入完全不可学”，而是因为网络没有显式建模大小与深度之间的结构关系。换言之，当模型面对一个强主效应（size）和一个弱主效应（depth）时，若仍采用普通共享 depth head，深度判别将被更强的大小和强度信息淹没。

### 3.4 引入 size-routed depth experts 后，原始输入神经网络可以恢复 coarse depth 信息
当单一 depth head 被替换为 size-routed depth experts 后，原始输入深度网络的性能显著改善。在 ground-truth size route 条件下，该模型的测试平衡准确率达到 0.5238，已经与 XGBoost 的 0.5138 基本相当。这是本文最关键的结果之一：它说明粗深度信息确实可以直接从原始时空触觉窗口中学习得到，但前提是网络必须尊重 depth 的 size-dependent 结构。

### 3.5 latent probes 说明网络内部编码了超出 size 的深度相关结构
为了验证 raw-input size-routed 深度网络内部是否真的学习到了具有物理意义的深度敏感信息，本文冻结训练后的网络 trunk，并在其潜在表示上训练线性 ridge probes 去恢复前期分析中识别出的结构化特征。结果显示，latent probe 的平均测试 `R^2` 为 0.3657，而仅以 size identity 为输入的 probe 为 0.2356，前者明显更高。尤其在 distribution complexity、temporal phase 和 deformation position 等特征家族上，latent representation 的可恢复性明显优于 size-only probe。这说明网络内部并不是简单“记住了大小”，而是已经编码了一部分深度相关结构。

### 3.6 hard-pair 分析显示模型并非只依赖峰值幅值，但对难例的分离仍有限
为了进一步检验网络是否会把“大而深”误判成“小而浅”，本文构造了若干 hard-pair 样本对，在这些样本中，浅层小结节和深层大结节在峰值强度上具有一定相似性，但其扩散、复杂度和时序结构不同。结果显示，更深样本在约 75% 的 hard-pair 对中获得更高的 `p_deep`，说明模型并非单纯依赖峰值幅值进行判断。但另一方面，由于这些样本对在部分 amplitude 量上仍存在残余差异，因此当前证据更适合写成“中等强度支持”，而不能过度宣称模型已完全解决大深/小浅混淆。

### 3.7 phase occlusion 表明当前网络仍偏依赖峰值邻域
phase occlusion 分析进一步揭示了网络的时间策略。遮挡 `peak neighborhood` 后，真实深度类别概率的平均下降幅度最大；而遮挡 `loading early` 和 `release` 阶段造成的下降相对较小。这与前面的机制分析并不完全一致，因为非深度学习分析提示早期加载与释放阶段更可能携带深度相关信息。由此可见，当前 raw-input 深度网络已经学到了一部分正确的深度结构，但其时序注意力仍然过度集中在峰值邻域，这也是后续模型迭代的重要方向。

### 3.8 真实部署链路中的主要误差来自 size routing，而非 depth trunk 本身
虽然 raw-input size-routed 深度模型在 oracle route 条件下性能良好，但真实部署必须依赖预测 size route。实验发现，若使用早期共享多任务 size head 进行路由，Stage3 正窗上的 size top-1 仅约 0.5226，从而使 predicted-route 深度平衡准确率下降至 0.4822。进一步分析显示，当 size route 正确时，depth 的平衡准确率可达 0.5557；而当 route 错误时，该指标降至 0.3204。由此可以清楚判断，当前链路中的主要误差传播源是上游 size router，而不是下游 depth trunk 本身。

### 3.9 专用 size router 与统一层级反演器进一步改善了真实链路性能
针对上述瓶颈，本文训练了专用 raw-input size router，并进一步构建了统一层级反演器。专用 size router 在测试集阳性窗口上的 top-1 为 0.6600，top-2 为 0.8405。进一步地，统一层级反演器在 predicted-route depth 条件下取得 0.5337 的平衡准确率，不仅超过旧 predicted-route 链路，也超过了 XGBoost 的 0.5138。这说明：若论文评价目标从“单点 size 指标最优”转向“真实部署链路中的深度判别鲁棒性”，统一层级反演器是更合理的最终系统。

## 4 讨论
本文的主要结论可以概括为三层。首先，从机制分析和 XGBoost 基线可知，深度并非不可学，而是一个显著弱于大小、且在不同大小条件下表现不同的信号。其次，从 raw-input 神经网络的对比实验可知，若网络结构不显式处理 size-depth 耦合关系，深度学习会退化到多数类附近；而一旦使用 size-routed 专家结构，原始输入即可学出可用的 coarse depth。最后，从真实部署链路的 predicted-route 评估可知，系统的关键不只是提升 depth trunk 本身，而是减少 size route 误差向 depth 的传导，这也是统一层级反演器优于松耦合串联方案的根本原因。

需要指出的是，本文当前结果仍有几个边界。其一，统一层级反演器虽然在 predicted-route depth 上取得了最佳结果，但其单独的 size 指标尚未在所有维度上全面超过 XGBoost；其二，hard-pair 与 phase occlusion 分析提示模型已经具备部分深度可解释性，但对 loading/release 阶段的利用仍不足；其三，本文的深度任务目前仍定位于 coarse depth classification，而非更细粒度的连续深度反演。因此，在后续工作中，需要进一步强化 phase-aware 建模、引入更针对路由噪声的训练策略，并推动由粗深度向更细粒度深度推断的扩展。

## 5 结论
本文围绕术中肺结节定位中的触觉缺失问题，构建了一套基于柔性阵列触觉传感与层级时空神经网络的定位与特征反演系统。研究结果表明：结节大小是表面应力响应的强主效应，而深度则是较弱但可学习的条件依赖效应；粗深度信息并非无法从原始触觉数据中恢复，而是必须通过大小感知路由和层级结构设计来显式建模；在真实部署链路中，统一层级反演器已将 predicted-route depth 性能稳定提升至超过 XGBoost 基线的水平。更重要的是，本文通过机制分析、结构化基线和神经网络解释验证形成了较完整的证据链，使“模型为什么能判别深度”这一问题获得了可检验而非纯黑箱的回答。该研究为柔性触觉传感在术中肺结节定位中的应用提供了一个可部署、可解释、可进一步迭代的技术基础。
